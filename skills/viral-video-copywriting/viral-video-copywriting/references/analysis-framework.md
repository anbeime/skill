## 概览
本框架提供了一套系统化的爆款短视频分析维度，帮助深度拆解视频成功的关键因素，为后续文案创作提供可复用的规律和策略。

## 开头钩子分析

### 分析要点
**时间范围**：前3-5秒
**核心问题**：如何在最短时间内抓住观众注意力？

### 钩子类型分类

1. **悬念型**
   - 抛出反常现象
   - 提出未解之谜
   - 预告惊人结论

2. **痛点型**
   - 直击用户困扰
   - 揭示普遍问题
   - 激发情感共鸣

3. **利益型**
   - 承诺明确价值
   - 展示吸引人的结果
   - 给出具体好处

4. **反转型**
   - 建立预期后打破
   - 制造认知冲突
   - 意外转折

5. **共鸣型**
   - 描述熟悉场景
   - 调动共同记忆
   - 引发情感认同

### 判断标准

- 吸引强度：_______（1-10分）
- 与主题关联度：_______（1-10分）
- 可复用性：_______（高/中/低）

## 文案结构分析

### 常见结构类型

1. **问题-解决方案结构**
   - 提出问题 → 分析原因 → 给出方案 → 展示效果

2. **对比结构**
   - 错误做法 vs 正确做法
   - 传统方式 vs 创新方式
   - 糟糕结果 vs 优质结果

3. **故事结构**
   - 背景 → 冲突 → 转折 → 解决 → 启示

4. **清单结构**
   - 要点1 → 要点2 → 要点3 → 总结

5. **递进结构**
   - 现象 → 原理 → 方法 → 实践 → 进阶

## 分析原则

1. **客观性**：基于实际内容进行分析，避免主观臆断
2. **系统性**：从多个维度全面分析，避免片面判断
3. **实用性**：提取可复用的规律，避免空洞理论
4. **差异化**：识别独特之处，避免泛泛而谈
5. **可操作性**：提供具体可行的建议，避免模糊指导

## 分析步骤
1. 观看视频3-5遍，整体感知
2. 逐帧分析开头3秒，识别钩子
3. 拆解文案结构，标记关键节点
4. 分析表达风格，记录特色表达
5. 定位核心价值，明确目标受众
6. 提取爆款要素，排序并评估
7. 形成分析报告，给出具体建议
