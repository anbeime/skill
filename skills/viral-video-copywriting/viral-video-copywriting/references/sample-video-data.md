# 示例视频数据

本文件提供完整的示例视频数据，用于测试和演示爆款短视频文案创作的完整流程。

## 示例1：知识科普类爆款视频

### 视频基本信息
- **标题**：3个小技巧让你工作效率翻倍
- **创作者**：职场成长记
- **视频时长**：45秒
- **数据表现**：点赞120万、评论3万、收藏50万

### 视频描述
```
职场必备技能，建议收藏！

很多人每天工作都很忙，但效率很低。今天分享3个简单实用的小技巧，帮你快速提升工作效率，告别加班！
#职场 #效率 #工作技巧
```

### 视频文案（字幕/口播）
```
你知道吗？90%的人都在用错误的方式工作。

今天分享3个小技巧，让你的工作效率翻倍。

第一个技巧：番茄工作法。设定25分钟专注工作，然后休息5分钟。这样能让你保持高度专注，避免疲劳。

第二个技巧：二八法则。找出最重要的20%任务，它们能带来80%的成果。把80%的精力放在这些任务上。

第三个技巧：每天下班前10分钟，写下明天的3件最重要的事。这样第二天一上班，你就知道该做什么，不会浪费时间。

学会了吗？收藏起来，明天就开始用！
```

### 背景音乐信息
- **音乐类型**：轻快电子音乐
- **风格特点**：节奏明快，积极向上
- **BPM**：约120
- **适合场景**：知识分享、正能量内容

### 标签
- 职场
- 效率
- 工作技巧
- 个人成长
- 时间管理

## 使用方法

### 作为测试数据
1. 选择任意一个示例
2. 复制视频标题、描述、文案等信息
3. 在第一步中提供这些信息（方式B：手动提供）
4. 继续执行后续流程

### 作为演示数据
1. 向用户展示这些示例
2. 说明整个流程如何运行
3. 让用户理解每一步的输入和输出

### 作为参考数据
1. 用户可以参考这些示例的视频信息格式
2. 当用户使用方式B时，按照这个格式提供信息
3. 确保信息的完整性和一致性
