# 小红书美妆内容创作技能

## 任务目标
- 本技能用于：为美妆护肤产品创建小红书平台的高质量笔记内容和配图提示词
- 能力包含：
  1. 笔记生成：根据商品信息、卖点、目标用户生成符合小红书风格的内容笔记
  2. 笔记优化：优化已有笔记的标题、正文、标签，提升吸引力和传播力
  3. 文生图描述：为美妆内容生成专业的AI绘画提示词
- 触发条件：用户提出小红书笔记创作需求、笔记优化需求，或需要为美妆内容生成配图提示词时

## 前置准备
- 无需特殊依赖包
- 无需非标准文件/文件夹

## 操作步骤

### 标准流程：笔记生成

1. **收集商品信息**
   - 询问用户提供：产品名称、品牌、价格、核心成分、主要功效、适合肤质、使用场景
   - 确定目标用户画像：年龄段、肤质、消费能力、需求痛点

2. **选择笔记类型**
   - 根据[references/note_templates.md](references/note_templates.md)选择合适的模板类型：
     - 产品测评型：适合产品详细测评
     - 护肤干货型：适合知识分享
     - 成分科普型：适合成分解析
     - 痛点解决型：适合解决具体问题
     - 对比测评型：适合多产品对比
     - 使用教程型：适合教程分享
     - 好物合集型：适合多产品推荐
     - 经验分享型：适合心得分享

3. **创作笔记内容**
   - **标题创作**：使用emoji、突出卖点、制造紧迫感（参考 note_templates.md 中的标题优化技巧）
   - **开头钩子**：制造共鸣、突出效果、给予承诺
   - **正文结构**：
     - 产品信息（名称、品牌、价格、规格）
     - 使用感受（质地、味道、上脸感、效果）
     - 核心卖点（功效、成分、适合人群）
     - 使用方法（步骤、注意事项）
     - 避雷提醒（不适合人群、使用禁忌）
     - 总结推荐（综合评价、性价比）
   - **标签组合**：根据[references/tag_library.md](references/tag_library.md)选择10-15个核心标签
   - **语气风格**：亲切自然、使用"姐妹们"、"家人们"等称呼、适当使用感叹词

4. **优化笔记质量**
   - 检查是否符合小红书风格：emoji使用、段落分隔、重点突出
   - 确保内容真实可信：避免过度夸大、提供真实感受
   - 突出产品卖点：核心功效、独特优势、性价比
   - 增加互动感：使用问句、引导评论、鼓励分享

### 标准流程：笔记优化

1. **分析原始笔记**
   - 识别问题点：标题吸引力不足、正文结构混乱、标签不精准
   - 评估改进空间：可优化的方向（标题、内容、标签）

2. **标题优化**
   - 添加emoji吸引眼球
   - 突出卖点（性价比、效果、适合人群）
   - 制造紧迫感（必入、闭眼入、无限回购）
   - 使用感叹词（绝绝子、太香了、好用到哭）

3. **正文优化**
   - 优化开头：制造共鸣、突出效果、给予承诺
   - 优化结构：使用emoji分段、数字标记、重点加粗
   - 优化内容：增加细节描述、提供真实感受、突出卖点
   - 优化结尾：总结推荐、引导互动

4. **标签优化**
   - 根据[references/tag_library.md](references/tag_library.md)重新选择标签
   - 确保标签与内容匹配
   - 控制标签数量（10-15个）
   - 结合热门标签

### 标准流程：文生图描述创作

1. **了解配图需求**
   - 确定图片用途：产品展示、人物肖像、美妆场景、氛围感图
   - 明确风格要求：温柔清新、精致奢华、青春活力、温柔治愈

2. **创作提示词**
   - 参考[references/image_prompt_guide.md](references/image_prompt_guide.md)中的提示词模板
   - 按照公式构建提示词：主体 + 描述 + 场景 + 风格 + 细节 + 构图
   - 选择合适的描述词库：
     - 皮肤描述：光泽皮肤、清透底妆、完美皮肤
     - 妆容描述：精致妆容、大地色眼影、豆沙色口红
     - 光线描述：自然光、柔和光线、晨光
     - 风格描述：温柔清新、精致奢华、青春活力
     - 构图描述：半身像、居中构图、背景虚化

3. **优化提示词**
   - 突出重点：把重要词放在前面
   - 细节丰富：描述质感、细节、光线
   - 风格明确：指定艺术风格、色调、氛围感
   - 场景具体：说明环境、元素、氛围

## 注意事项

- 内容真实性：避免过度夸大产品效果，提供真实使用感受
- 小红书风格：使用emoji、亲切语气、符合平台调性
- 标签精准度：根据笔记内容选择匹配的标签，避免无关标签
- 图文一致：文生图描述要与笔记内容、产品调性保持一致
- 避免敏感词：不使用绝对化用语、不涉及医疗功效、不夸大宣传
- 版权意识：描述中避免直接引用品牌Logo、商标

## 使用示例

### 示例1：产品测评型笔记生成
- **需求**：为某款美白精华创作小红书笔记
- **执行方式**：智能体自然语言创作
- **关键要点**：
  - 收集产品信息（价格、成分、功效）
  - 选择产品测评型模板
  - 创作标题："✨挖到宝了！这个美白精华真的绝绝子！相见恨晚😭"
  - 撰写正文（产品信息、使用感受、避雷提醒、总结）
  - 选择标签：#护肤推荐 #美白 #烟酰胺 #学生党 #平价好物
  - 优化排版（emoji分段、重点突出）

### 示例2：笔记优化
- **需求**：优化现有护肤笔记，提升吸引力
- **执行方式**：智能体自然语言优化
- **关键要点**：
  - 分析原始笔记问题点
  - 优化标题：添加emoji、突出卖点
  - 优化正文：改进结构、增加细节
  - 优化标签：重新选择精准标签
  - 检查排版和风格

### 示例3：文生图描述创作
- **需求**：为护肤品创作产品展示图提示词
- **执行方式**：智能体自然语言创作
- **关键要点**：
  - 确定图片用途（产品展示）
  - 参考产品展示提示词模板
  - 构建完整提示词（主体+描述+场景+风格+细节+构图）
  - 优化描述（突出质感、光线、风格）
  - 输出专业提示词供AI绘画使用

## 质量标准

- 笔记内容：
  - 标题吸引眼球，有emoji和卖点
  - 正文结构清晰，逻辑流畅
  - 内容真实可信，无过度夸大
  - 标签精准匹配，数量适中
- 文生图描述：
  - 主体明确，描述详细
  - 风格一致，氛围感强
  - 细节丰富，画面感强
  - 符合产品调性和平台风格
