## 微信

### 接口地址
```
http://39.108.254.228:8002/publish-draft
```

### 内容格式
- **支持格式**：HTML富文本
- **标题要求**：简洁吸引人，建议15-25字
- **封面比例**：2.35:1（推荐1600x680px）
- **封面大小**：不超过5MB

### 排版规范
- 结构分层：使用`<h1>`, `<h2>`, `<h3>`等标题标签
- 重点划线：使用`<mark>`标签，颜色规范：
  - 浅黄：`#fff3cd`
  - 浅绿：`#d4edda`
  - 浅红：`#f8d7da`
  - 浅蓝：`#e8f4f8`
  - 浅紫：`#f3e5f5`
  - 浅橙：`#fff8e1`
- 划线密度：不超过全文20%
- 纯净输出：不包含作者信息、制作信息

### 示例HTML
```html
<h1>文章标题</h1>

<p>文章正文内容...</p>

<p>重点内容使用<mark style="background-color: #fff3cd;">浅黄色划线</mark>标注。</p>

<h2>章节标题</h2>

<p>章节内容...</p>
```

## 接口配置说明

### 统一接口规范

**接口地址**：
```
http://39.108.254.228:8002/publish
```

**请求方法**：`POST`

**请求头**：
```
Content-Type: application/json
```

**请求体**：
```json
{
  "platform": "zhihu",
  "title": "文章标题",
  "content": "文章内容",
  "cover": "封面图URL（可选）",
  "digest": "摘要/标签（可选）"
}
```

**返回格式**：
```json
{
  "success": true,
  "message": "发布成功",
  "data": {
    "url": "文章链接",
    "id": "文章ID"
  }
}
```
