## 平台分类与特点

### 深度阅读类
**平台**：微信公众号、WordPress

**特点**：
- 内容长度：1500-3000字
- 内容风格：深度、干货、逻辑严密
- 排版要求：精美、结构分层、重点划线
- 标题风格：悬念、数字、疑问、观点鲜明
- 封面比例：微信2.35:1、WordPress 16:9

**适合内容**：
- 深度分析文章
- 系统教程
- 行业报告
- 技术白皮书

---

### 知识问答类
**平台**：知乎

**特点**：
- 内容长度：1000-2000字
- 内容风格：专业、理性、有观点、逻辑清晰
- 排版要求：Markdown格式，段落清晰
- 标题风格：问题式、观点式、包含核心关键词
- 封面比例：16:9

**适合内容**：

## 差异化内容策略

### 策略一：同一主题，多版本适配

根据平台特性，为同一主题生成不同版本的内容。

**示例主题：AI技术革命**

#### 微信公众号版本（深度版）
- **特点**：1500-3000字，HTML排版，结构分层+划线
- **标题**：AI技术革命：从效率到创新的三重变革
- **内容结构**：
  1. 引言：热点切入
  2. 第一重变革：效率革命
  3. 第二重变革：决策优化
  4. 第三重变革：创新加速
  5. 结论：行动建议
  6. 金句：AI不会取代人类，但会取代不会使用AI的人

#### 知乎版本（问答版）
- **特点**：1000-2000字，Markdown格式，逻辑清晰
- **标题**：AI技术革命：我们该如何应对？
- **内容结构**：
  1. 引言：提出问题
  2. 核心观点：AI不会取代人类，但会取代不会使用AI的人
  3. 理由一：效率提升
  4. 理由二：决策优化
  5. 理由三：创新加速
  6. 结论：拥抱变化
  7. 互动：欢迎评论区交流

#### 微博版本（短内容版）
- **特点**：140-280字，纯文本，话题标签
- **内容**：
  ```
  AI技术正在改变世界！🚀

  效率革命、决策优化、创新加速，AI无处不在。

  最重要的是：AI不会取代人类，但会取代不会使用AI的人。

  #AI技术 #科技前沿 #未来趋势
  ```

#### 小红书版本（图文版）
- **特点**：300-800字，emoji+标签，短段落
- **内容**：
  ```
  AI时代来了！你准备好了吗？🤖

  最近玩了很多AI工具，真的太香了！✨

  📌 核心观点：
  ✅ AI提升效率，处理重复工作
  ✅ AI辅助决策，减少偏差
  ✅ AI加速创新，催生新方案

  💡 建议：
  1. 学习使用AI工具
  2. 专注创造性工作
  3. 保持学习心态

  #AI #黑科技 #效率工具 #职场干货
  ```

#### 掘金版本（技术版）
- **特点**：800-1500字，Markdown格式，代码示例
- **标题**：一文读懂AI技术变革
- **内容结构**：
  1. 前言：本文介绍AI技术带来的三重变革
  2. 1. 效率革命（代码示例）
  3. 2. 决策优化（代码示例）
  4. 3. 创新加速
  5. 总结：拥抱AI，提升核心竞争力
  6. 标签：#AI #技术 #前沿

## 平台内容模板

### 微信公众号模板

**标题策略**：
- 数字型："3个AI工具提升10倍效率"
- 疑问型："AI会取代我们吗？"
- 对比型："传统方法 vs AI方法"
- 警告型："不注意这3点，将被AI淘汰"

**内容结构**：
```html
<h1>标题</h1>

<h2>引言：热点/痛点切入</h2>
<p>...</p>

<h2>第一部分：核心观点一</h2>
<p>...</p>

<h2>第二部分：核心观点二</h2>
<p>...</p>

<h2>第三部分：核心观点三</h2>
<p>...</p>

<blockquote>金句总结</blockquote>

<h2>结论：行动建议</h2>
<p>...</p>
```

**排版技巧**：
- 每段3-5句，保持简洁
- 使用`<mark>`标记关键信息（不超过20%）
- 颜色规范：浅黄#fff3cd、浅绿#d4edda、浅红#f8d7da等
- 不包含作者信息、制作信息

## 智能体适配规则

### 规则1：自动识别平台特性

智能体根据目标平台，自动应用对应的：
- 内容长度
- 内容格式
- 标题风格
- 封面比例
- 标签格式

### 规则2：保持核心信息一致

无论发布到哪个平台，核心观点和关键信息保持一致，只是表达方式和侧重点不同。

### 规则3：优化标题和封面

每个平台使用最优的标题和封面：
- 标题：符合平台风格，吸引目标用户
- 封面：符合平台比例，相关性强

### 规则4：适配标签格式

每个平台使用正确的标签格式：
- 微博/X：#话题# 格式
- 小红书：#标签 格式
- 知乎/掘金/CSDN：最多5个标签
- B站：1-10个标签

### 规则5：纯净输出

所有平台都不包含：
- 作者信息
- 制作信息（如"COZE制作"、"AI助手"）
- 推广信息（除非用户明确要求）

## 批量发布工作流

### 工作流步骤

1. **生成主内容**（以微信公众号为主）
   - 生成长深度内容（1500-3000字）
   - 使用HTML格式
   - 结构分层+划线

2. **适配知乎**（深度→问答）
   - 转换为Markdown格式
   - 调整为问答式结构
   - 缩短至1000-2000字

3. **适配微博**（深度→短内容）
   - 提取核心观点
   - 转换为纯文本
   - 缩短至140-280字
   - 添加话题标签

4. **适配小红书**（深度→图文）
   - 提取核心要点
   - 使用emoji+短段落
   - 缩短至300-800字
   - 添加标签

5. **适配掘金/CSDN**（深度→技术）
   - 转换为Markdown格式
   - 提取技术要点
   - 添加代码示例
   - 缩短至800-1500字

6. **批量发布**
   - 调用`publish.py`依次发布到各平台
   - 确保格式和参数正确

## 总结

差异化内容生成的核心是：

1. **理解平台特性**：每个平台都有独特的风格和用户群体
2. **保持核心一致**：核心观点和关键信息保持一致
3. **优化表达方式**：根据平台调整表达方式、长度、格式
4. **提升传播效果**：使用最优的标题、封面、标签
