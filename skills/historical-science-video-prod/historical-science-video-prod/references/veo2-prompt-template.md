# Veo2 提示词模板

## 标准格式

每条 Veo2 提示词必须包含以下 6 个要素：

---

### 主体（Subject）
- 视频的主要对象是什么
- 对象的特征和细节
- 对象的数量（如果是多个）

### 动作（Action）
- 场景中发生什么动作
- 动作的速度和节奏
- 动作的连续性

### 场景（Setting）
- 环境描述
- 背景元素
- 空间感

### 镜头角度（Camera Angle/Movement）
- 镜头类型（特写、远景等）
- 镜头运动（平移、推进等）
- 视角变化

### 灯光（Lighting）
- 光源类型
- 光线强度
- 光影效果

### 风格/情绪（Style/Mood）
- 视觉风格
- 情感氛围
- 艺术效果

---

## 复古风格固定参数

所有 Veo2 提示词必须包含以下复古风格参数：

```
风格/情绪（Style/Mood）：
- 视觉风格：棕褐色调，复古历史照片质感，轻微褪色效果，细腻胶片颗粒感，柔和阴影
- 情感氛围：根据文案情绪调整（悬念/严肃/激昂/沉静）
- 艺术效果：历史真实感与档案感拉满，无任何跨年代元素
```

## 示例提示词

### 远景镜头示例

```
主体：1900年伦敦黎明全景，泰晤士河河面（泛着微光，雾气厚重），2辆木质马车（车身有复古雕花，车轮沾有湿润泥土），鹅卵石街道（湿润反光，有细微车辙），国会大厦（背景建筑，轮廓在雾气中隐约可见）

动作：马车缓慢行驶（速度均匀，车轮滚动无扬尘），雾气缓慢流动（轻柔扩散）

场景：1900年伦敦市区，河面与街道衔接自然，远处建筑错落有致，空间层次分明

镜头角度：远景，镜头从左向右缓慢平移（速度平缓，无抖动）

灯光：自然光（黎明微光，光线柔和，无强烈阴影），雾气折射光线形成朦胧感

风格/情绪：
- 视觉风格：棕褐色调，复古历史照片质感，轻微褪色效果，细腻胶片颗粒感，柔和阴影
- 情感氛围：沉静肃穆
- 艺术效果：历史真实感与档案感拉满，无任何20世纪后元素
```

### 人物中景示例

```
主体：约瑟夫·李斯特，58岁男性，留着19世纪末英式络腮胡，身着深棕色羊毛实验外套（袖口沾有微量实验试剂痕迹），领口系白色亚麻领巾，眼神专注略带凝重

动作：站在实验窗前，手持培养皿，目光聚焦在培养皿内部

场景：19世纪末英国实验室，窗外是伦敦雾景（虚化处理），实验台上有显微镜、玻璃器皿

镜头角度：中景，镜头缓慢推进（速度平缓，聚焦人物神情）

灯光：柔和侧光（透过窗户的自然光），面部细节清晰，有立体感

风格/情绪：
- 视觉风格：棕褐色调，复古历史照片质感，轻微褪色效果，细腻胶片颗粒感，柔和阴影
- 情感氛围：专注、凝重
- 艺术效果：历史真实感与档案感拉满，无任何现代元素
```

### 实验特写示例

```
主体：培养皿内链球菌菌落（白色圆形菌落，分布均匀），培养皿玻璃反光清晰，背景虚化

动作：菌落静止无动作，培养皿缓慢转动（展示不同角度）

场景：实验室暗淡环境，聚焦于培养皿细节

镜头角度：特写，镜头从上方缓慢旋转（展示菌落全貌）

灯光：柔和顶光（实验室灯光），菌落纹理清晰，玻璃反光自然

风格/情绪：
- 视觉风格：棕褐色调，复古历史照片质感，轻微褪色效果，细腻胶片颗粒感，柔和阴影
- 情感氛围：神秘、科学
- 艺术效果：历史真实感与档案感拉满，无任何现代元素
```

## 提示词生成原则

1. **具体化描述**：使用清晰、具体的描述，避免模糊或抽象的表达
2. **逻辑连贯性**：保持提示词内部的逻辑性和连贯性
3. **避免矛盾**：避免模糊或矛盾的描述
4. **涵盖全要素**：确保涵盖所有必要的视觉元素
5. **适配分镜**：每条提示词严格对应一个分镜的内容
