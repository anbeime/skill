# 文案结构说明

## 总时长分配

3分钟视频的总时长分配如下：

| 部分 | 时长 | 字数 | 内容要点 |
|------|------|------|----------|
| 开篇悬念 | 20秒 | 约30-40字 | 用[年份]+[科学事件/人物困惑]开头 |
| 历史背景 | 30秒 | 约45-55字 | 交代时代、地点、学术或社会背景 |
| 人物描写 | 25秒 | 约35-45字 | 突出科学家的特质 |
| 事件与实验细节 | 45秒 | 约65-80字 | 数字化描写实验对象、方法、数量 |
| 发现与突破 | 35秒 | 约50-60字 | 明确科学结论或规律 |
| 意义与影响 | 30秒 | 约45-55字 | 解释科学发现如何改变学界 |
| 结尾反思 | 15秒 | 约20-30字 | 提出深思或未来探索问题 |
| **总计** | **200秒** | **约900字** | |

---

## 各部分详细说明

### 1. 开篇悬念（20秒）

**目标**：在前 8 秒抓住观众注意力

**写法**：
- 用具体年份开头（如"1961年冬天"）
- 描述一个惊人的科学事件或人物困惑
- 使用具体数字或场景
- 语气坚定有力

**示例**：
```
"1961年冬天，麻省理工学院的一间实验室里，一次看似普通的四舍五入，让整个科学界震惊：0.000127的微小误差，竟能让未来的天气预报彻底失灵。"
```

---

### 2. 历史背景（30秒）

**目标**：交代时代、地点、学术或社会背景

**写法**：
- 说明当时的科学环境
- 描述社会或学术氛围
- 为后续故事做铺垫

**示例**：
```
"20世纪60年代的美国，正值计算机科学蓬勃发展的时期。气象学家们试图用数学模型来预测天气，认为只要掌握了足够多的数据，就能精确计算出未来的天气变化。"
```

---

### 3. 人物描写（25秒）

**目标**：突出科学家的特质

**写法**：
- 描述科学家的人物特点
- 突出坚持、偏执、热情、天才等特质
- 为后续实验故事做铺垫

**示例**：
```
"爱德华·洛伦茨，这位兼具数学家和气象学家双重身份的科学家，以近乎偏执的严谨著称。他每天都在反复运行天气模拟程序，试图解开天气预报的终极谜题。"
```

---

### 4. 事件与实验细节（45秒）

**目标**：数字化描写实验对象、方法、数量、时间、空间

**写法**：
- 使用具体数字（年份、数量、时间）
- 描述实验过程和方法
- 突出实验的严谨性

**示例**：
```
"洛伦茨按下回车键，开始重新运行程序。他原本以为，这次简化计算的结果会和之前差不多——毕竟，千分之一级别的误差，在整个复杂的天气系统中，根本不值一提。
第一个月，两条曲线还紧密贴合，就像双胞胎一样同步变化。但从第二个月开始，诡异的事情发生了——两条曲线开始产生微小的分歧。然后分歧越来越大。到了第三个月，它们已经完全变成了两条不同的曲线，代表的天气状态天差地别。"
```

---

### 5. 发现与突破（35秒）

**目标**：明确科学结论或规律，用生动语言强调其重要性

**写法**：
- 清晰阐述科学发现
- 用生动的语言描述发现的意义
- 强调对科学界的影响

**示例**：
```
"洛伦茨把这个现象总结为一个震撼世界的理论：混沌理论。它的核心是——复杂的非线性系统对初始条件极其敏感。简单来说，在复杂的系统中，初始状态的微小变化，会随着时间的推移被不断放大，最终导致巨大的、不可预测的结果。"
```

---

### 6. 意义与影响（30秒）

**目标**：解释科学发现如何改变学界、社会或文化

**写法**：
- 说明发现对科学界的改变
- 描述对社会或文化的影响
- 突出发现的重要性

**示例**：
```
"蝴蝶效应的发现，彻底改变了科学界对复杂系统的认知。它告诉我们：第一，天气预报有极限。即使今天的技术再先进，能预测的准确范围也就在两周左右。
第二，复杂系统本质上不可长期预测。从气候到股市，从生态系统到社会趋势，凡是涉及复杂互动的系统，长期预测都存在根本性的局限。"
```

---

### 7. 结尾反思（15秒）

**目标**：提出深思或未来探索问题

**写法**：
- 提出深刻的反思
- 或提出未来探索的问题
- 留给观众思考空间

**示例**：
```
"蝴蝶效应不仅是科学理论，更是人生的隐喻。你今天读的一本书、遇见的一个人、做出的一次选择，可能就是那只扇动翅膀的蝴蝶。在未来的某一天，它会引发一场属于你自己的龙卷风。"
```

---

## 语言特点

### 短句与长句交替
- 短句用于强调关键信息
- 长句用于描述细节和背景

### 口语化、生活化
- 使用日常用语
- 避免过于学术化的表达

### 保留科学准确性
- 数字、年份、数据准确
- 科学概念表述正确

### 多用数字、年份、数量
- 增强真实感
- 增强说服力

### 适当加入比喻或类比
- 让抽象概念易理解
- 增强表达生动性

---

## 注意事项

1. **时长控制**：每个部分严格控制时长，总时长不超过 3 分钟
2. **节奏紧凑**：避免冗余铺垫，每句话都服务于内容推进
3. **情绪递进**：从悬念到探索，到发现，到反思，情绪要有层次感
4. **口播友好**：考虑语音朗读的节奏和停顿
