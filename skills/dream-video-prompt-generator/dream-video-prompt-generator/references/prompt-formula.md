## 1. 公式概述

### 核心公式

**中文版公式**：
```
主体（主体描述）+ 运动 + 场景（场景描述）+（镜头语言 + 光影 + 氛围）
```

**英文版公式**：
```
Subject (Subject description) + Movement + Scene (Scene description) + (Shot language + Light + Atmosphere)
```

### 公式设计原则

1. **完整性**：涵盖主体、动作、环境、视觉、情绪所有关键要素
2. **顺序性**：按照逻辑顺序排列，便于AI理解和生成
3. **简洁性**：使用简洁明确的语言，避免冗余
4. **适配性**：适配即梦等视频生成AI工具
5. **可扩展性**：可根据需求补充延伸细节

## 2. 公式拆解

### 2.1 主体（Subject）

**定义**：描述数字人角色的外貌、服饰、气质等核心特征

**关键要素**：
- 基本信息：性别、年龄
- 外貌特征：脸型、眼睛、发型、肤色、唇色等
- 服饰描述：上衣、下装、鞋子、配饰等
- 气质特征：专业、亲切、热情、权威等

**小省导购员固定描述**：
```
女性数字人，25岁左右，鹅蛋脸、杏眼带笑、浅棕色齐肩卷发，肤色白皙，唇色淡粉；
身穿浅灰色修身西装套裙，内搭白色衬衫，脚踩米色细跟鞋，佩戴银色简约项链，
气质专业又亲和，手部姿态优雅自然。
```

**英文版**：
```
Female digital human, 25 years old, oval face, almond eyes with smile,
light brown shoulder-length curly hair, fair skin, light pink lips;
wearing light gray tailored suit skirt, white shirt, beige pumps,
simple silver necklace, professional and friendly temperament,
elegant hand gestures.
```

### 2.2 运动（Movement）

**定义**：描述数字人的具体动作

**关键要素**：
- 动作类型：抬手、点头、指向、转身等
- 动作速度：缓慢、快速、匀速等
- 动作方向：向上、向下、向左、向右等
- 动作幅度：轻微、明显、大幅等

**适配5s短视频的运动**：
- 缓慢抬手指向右侧
- 点头示意
- 手部手势展示
- 轻微转身
- 姿态调整

**示例**：
```
中文：缓慢抬手指向右侧，姿态挺拔
英文：Slowly raise hand pointing to right, upright posture
```

### 2.3 场景（Scene）

**定义**：描述数字人所在的环境

**关键要素**：
- 场景类型：商务场景、产品展示、促销场景等
- 背景元素：大屏、货架、书架、办公桌等
- 前景元素：文件、产品、道具等
- 空间关系：背景、中景、前景

**示例**：
```
中文：现代商务会议室，背景模糊大屏投影，前景摆简约文件
英文：Modern business meeting room, blurred large screen projection in background, simple documents in foreground
```

### 2.4 镜头语言（Shot Language）

**定义**：描述镜头的景别、运镜方式、构图等

**关键要素**：
- 景别：中景、近景、全景、特写
- 运镜方式：固定镜头、摇镜、推拉镜头
- 构图比例：9:16竖版（适配短视频平台）
- 视角：侧光、顺光、逆光

**适配5s短视频的镜头语言**：
- 中景固定镜头：展示人物上半身和部分环境
- 近景固定镜头：聚焦人物面部表情
- 轻微摇镜：跟随人物动作

**示例**：
```
中文：中景固定镜头，背景虚化，顺光拍摄
英文：Medium fixed shot, bokeh background, front light
```

### 2.5 光影（Light）

**定义**：描述光线的类型、方向、色调、明暗对比

**关键要素**：
- 光线类型：柔光、强光、侧光、逆光
- 光线方向：顺光、侧光、逆光
- 色调：暖调、冷调、中性
- 明暗对比：明显、适中、柔和

**适配情绪的光影**：
- 热情情绪：暖调柔光，明亮清晰
- 专业情绪：冷调强光，明暗对比明显
- 亲切情绪：暖调柔光，温暖柔和
- 紧迫情绪：冷调强光，对比强烈

**示例**：
```
中文：暖调柔光，明亮清晰
英文：Warm soft light, bright and clear
```

### 2.6 氛围（Atmosphere）

**定义**：描述整体的情绪氛围

**关键要素**：
- 情绪基调：热情、专业、亲切、紧迫、威严等
- 氛围感受：吸引、神秘、紧张、温暖、冷静等
- 情绪递进：从第一幕到第五幕的情绪变化

**适配五幕的情绪递进**：
- 第一幕：开场吸引（热情、神秘）
- 第二幕：产品介绍（专业、详细）
- 第三幕：对比分析（客观、冷静）
- 第四幕：促销优惠（紧迫、热烈）
- 第五幕：行动召唤（鼓励、坚决）

**示例**：
```
中文：热情专业，吸引注意力
英文：Enthusiastic and professional, attention-grabbing
```

## 5. 使用步骤

### Step 1：确定主体描述
- 使用核心角色固定描述（小省导购员）
- 确保所有分镜主体描述一致

### Step 2：设计运动
- 根据场景需求设计具体动作
- 动作要连贯，适配5s短视频
- 动作要简单明确，便于AI生成

### Step 3：描述场景
- 确定场景类型（商务、产品展示等）
- 描述背景和前景元素
- 确保场景连贯

### Step 4：设定镜头语言
- 选择合适的景别（中景、近景等）
- 确定运镜方式（固定镜头、摇镜等）
- 保持9:16竖版构图

### Step 5：配置光影
- 根据情绪基调选择光影类型
- 暖调适配热情，冷调适配专业
- 确保光影描述清晰

### Step 6：定义氛围
- 确定整体情绪基调
- 确保情绪递进自然
- 中英文版本语义一致

### Step 7：组合完整提示词
- 按公式顺序组合各要素
- 生成中文版完整提示词
- 生成英文版完整提示词
- 确保中英文语义对应

## 注意事项

1. **主体一致性**：所有分镜使用相同的主体描述
2. **动作连贯性**：动作要连贯，适配5s短视频
3. **场景连贯性**：场景元素要连贯，背景统一
4. **情绪递进**：从第一幕到第五幕情绪递进自然
5. **中英文对应**：中英文版本语义一致，术语准确
6. **细节具体**：动作和场景描述要具体，便于AI生成
7. **公式严格**：严格遵循公式顺序，不随意调整
