## 1. 概述

### 目的
确保五幕提示词在人物、视觉、情绪三个维度保持高度一致，规避AI独立生成的上下文断层问题，提升视频生成质量。

### 一致性三大维度
1. **人物一致性**：所有分镜中角色外貌、服饰、气质描述统一
2. **视觉连贯性**：色调、场景元素、构图比例贯穿五幕
3. **情绪连贯性**：从第一幕到第五幕的情绪递进自然

### 适用范围
- 即梦视频提示词生成
- Suno音乐提示词生成
- AI绘画提示词生成
- 跨工具适配

## 2. 人物一致性

### 2.1 核心原则
所有五幕提示词必须使用**完全相同**的：
- 外貌描述（脸型、眼睛、发型、肤色、唇色）
- 服饰描述（上衣、下装、鞋子、配饰）
- 气质描述（专业、亲切、热情等）
- 手部姿态描述（优雅自然、活跃等）

### 2.2 小省导购员固定描述（严禁修改）

**中文版**：
```
女性数字人，25岁左右，鹅蛋脸、杏眼带笑、浅棕色齐肩卷发，肤色白皙，唇色淡粉；
身穿浅灰色修身西装套裙，内搭白色衬衫，脚踩米色细跟鞋，佩戴银色简约项链，
气质专业又亲和，手部姿态优雅自然。
```

**英文版**：
```
Female digital human, 25 years old, oval face, almond eyes with smile,
light brown shoulder-length curly hair, fair skin, light pink lips;
wearing light gray tailored suit skirt, white shirt, beige pumps,
simple silver necklace, professional and friendly temperament,
elegant hand gestures.
```

### 2.3 人员一致性检查清单

#### 外貌一致性
- [ ] **性别一致**：所有分镜都描述为"女性数字人"/"Female digital human"
- [ ] **年龄一致**：所有分镜都描述为"25岁左右"/"25 years old"
- [ ] **脸型一致**：所有分镜都描述为"鹅蛋脸"/"oval face"
- [ ] **眼睛一致**：所有分镜都描述为"杏眼带笑/专注/兴奋"/"almond eyes with smile/focused/excited"
- [ ] **发型一致**：所有分镜都描述为"浅棕色齐肩卷发"/"light brown shoulder-length curly hair"
- [ ] **肤色一致**：所有分镜都描述为"肤色白皙"/"fair skin"
- [ ] **唇色一致**：所有分镜都描述为"唇色淡粉"/"light pink lips"

#### 服饰一致性
- [ ] **上衣一致**：所有分镜都描述为"浅灰色修身西装套裙，内搭白色衬衫"/"light gray tailored suit skirt, white shirt"
- [ ] **鞋子一致**：所有分镜都描述为"脚踩米色细跟鞋"/"beige pumps"
- [ ] **配饰一致**：所有分镜都描述为"佩戴银色简约项链"/"simple silver necklace"

#### 气质一致性
- [ ] **气质类型一致**：所有分镜都描述为"气质专业又亲和"/"professional and friendly temperament"
- [ ] **手部姿态一致**：所有分镜都描述为"手部姿态优雅自然"/"elegant hand gestures"

### 2.4 常见错误与修正

#### 错误示例1：外貌描述不一致
❌ **第一幕**："鹅蛋脸、杏眼带笑"
❌ **第二幕**："圆脸、大眼睛"
✅ **修正**：所有分镜都使用"鹅蛋脸、杏眼带笑"

#### 错误示例2：服饰描述不一致
❌ **第一幕**："身穿浅灰色修身西装套裙"
❌ **第二幕**："穿着蓝色职业装"
✅ **修正**：所有分镜都使用"身穿浅灰色修身西装套裙"

#### 错误示例3：气质描述不一致
❌ **第一幕**："气质专业又亲和"
❌ **第三幕**："气质严肃冷漠"
✅ **修正**：所有分镜都使用"气质专业又亲和"，通过光影和氛围调整严肃感

## 3. 视觉连贯性

### 3.1 核心原则
设定贯穿五幕的视觉线索，保持色调、场景元素、构图比例统一。

### 3.2 视觉线索设计

#### 色调连贯
- **主色调**：浅灰商务色调（适配专业导购场景）
- **辅助色调**：根据情绪调整（暖调适配热情，冷调适配专业）
- **色彩饱和度**：保持适中，避免过饱和或过淡

**示例**：
```
第一幕：浅灰商务色调 + 暖调柔光（热情专业）
第二幕：浅灰商务色调 + 暖调柔光（专业详细）
第三幕：浅灰商务色调 + 冷调柔光（客观冷静）
第四幕：浅灰商务色调 + 暖调强光（紧迫热烈）
第五幕：浅灰商务色调 + 暖调柔光（鼓励坚决）
```

#### 场景元素连贯
- **背景统一**：背景场景类型统一（商务场景）
- **道具一致**：道具（平板、文件、产品）描述一致
- **空间关系**：前景、中景、背景关系统一

**示例**：
```
第一幕：现代商务场景，背景产品展示架，前景简约文件
第二幕：现代商务场景，背景产品展示架，前景产品介绍页
第三幕：现代商务场景，背景产品对比表，前景价格标签
第四幕：现代商务场景，背景促销横幅，前景特价商品
第五幕：现代商务场景，背景购买引导，前景购物车图标
```

#### 构图比例连贯
- **比例统一**：9:16竖版（适配短视频平台）
- **景别统一**：以中景为主，近景为辅
- **视角统一**：以顺光、侧光为主，避免频繁切换视角

### 3.3 视觉连贯性检查清单

- [ ] **色调连贯**：主色调贯穿五幕
- [ ] **背景统一**：背景场景类型统一
- [ ] **道具一致**：道具描述一致
- [ ] **比例统一**：9:16竖版构图
- [ ] **景别合理**：景别切换合理
- [ ] **视角统一**：视角切换不频繁

### 3.4 常见错误与修正

#### 错误示例1：色调不连贯
❌ **第一幕**："浅灰商务色调，暖调柔光"
❌ **第三幕**："深蓝色商务色调，冷调强光"
✅ **修正**：保持"浅灰商务色调"，通过光影调整情绪

#### 错误示例2：背景不统一
❌ **第一幕**："现代商务会议室"
❌ **第三幕**："户外草坪场景"
✅ **修正**：保持"现代商务场景"

#### 错误示例3：构图比例不统一
❌ **第一幕**："9:16竖版"
❌ **第三幕**："16:9横版"
✅ **修正**：统一使用"9:16竖版"

## 4. 情绪连贯性

### 4.1 核心原则
从第一幕到第五幕的情绪递进自然，符合场景逻辑和用户体验。

### 4.2 情绪递进设计

#### 产品推荐场景情绪递进
```
第一幕（开场吸引）：热情、神秘
第二幕（产品介绍）：专业、详细
第三幕（对比分析）：客观、冷静
第四幕（促销优惠）：紧迫、热烈
第五幕（行动召唤）：鼓励、坚决
```

#### 价格对比场景情绪递进
```
第一幕（开场引导）：亲切、好奇
第二幕（平台介绍）：客观、清晰
第三幕（价格对比）：理性、冷静
第四幕（优势分析）：公正、平衡
第五幕（购买建议）：鼓励、友好
```

#### 促销活动场景情绪递进
```
第一幕（活动开场）：兴奋、热烈
第二幕（优惠介绍）：紧迫、刺激
第三幕（限时秒杀）：急促、激烈
第四幕（库存告急）：危急、催促
第五幕（最后机会）：果断、坚定
```

### 4.3 情绪一致性检查清单

- [ ] **情绪基调一致**：整体情绪基调一致（如专业、热情）
- [ ] **情绪递进自然**：从第一幕到第五幕情绪递进自然
- [ ] **情绪变化合理**：情绪变化符合场景逻辑
- [ ] **情绪适配光影**：光影适配情绪（暖调适配热情，冷调适配专业）
- [ ] **氛围描述统一**：氛围描述与情绪一致

### 4.4 常见错误与修正

#### 错误示例1：情绪跳跃过大
❌ **第一幕**："热情、神秘"
❌ **第二幕**："严肃、冷漠"
✅ **修正**：第一幕"热情、神秘"，第二幕"专业、详细"

#### 错误示例2：情绪不递进
❌ **所有五幕**："热情、神秘"
✅ **修正**：情绪递进：热情→专业→客观→紧迫→鼓励

#### 错误示例3：情绪与光影不匹配
❌ **情绪描述**："热情、热烈"
❌ **光影描述**："冷调强光"
✅ **修正**：使用"暖调强光"适配热烈情绪

## 5. 质量核查流程

### 5.1 核查时机
- 提示词生成师生成五幕提示词后
- 质量管控师进行统一核查

### 5.2 核查顺序
1. **人物一致性核查**：确保角色描述统一
2. **视觉连贯性核查**：确保色调、场景、构图统一
3. **情绪连贯性核查**：确保情绪递进自然
4. **中英文对应核查**：确保中英文语义一致

### 5.3 核查结果处理

#### 通过（Pass）
- 所有关键项检查通过
- 直接传递给知识库运维师归档
- 传递给跨环节适配师生成跨工具提示词

#### 不通过（Fail）
- 识别具体问题
- 生成问题清单
- 反馈给提示词生成师调整
- 最多反馈2次，避免过度迭代

## 注意事项

1. **严格遵循固定描述**：小省导购员角色描述严禁修改
2. **色调贯穿始终**：主色调必须贯穿五幕
3. **情绪递进自然**：情绪变化要符合场景逻辑
4. **中英文对应**：中英文版本语义必须一致
5. **问题及时反馈**：发现不一致问题及时反馈给提示词生成师
6. **避免过度迭代**：最多反馈2次，避免过度迭代
