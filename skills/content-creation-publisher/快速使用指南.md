# 内容创作与发布全流程 - 快速使用指南

## 🎯 技能概述

**名称**：content-creation-publisher
**功能**：从内容采集到多平台发布的一站式解决方案
**位置**：`D:\tool\skills\content-creation-publisher\`

---

## 📦 包含的子技能

### 1. baoyu-url-to-markdown
- **功能**：网页转Markdown
- **技术**：Chrome CDP
- **用途**：采集文章、论文、新闻

### 2. baoyu-format-markdown
- **功能**：Markdown格式化
- **特点**：中英文间距、标点修正、加粗bug修复
- **用途**：优化文档排版

### 3. article-illustrator
- **功能**：智能文章配图
- **特点**：6种类型 × 8种风格
- **用途**：自动生成插图

### 4. baoyu-post-to-wechat
- **功能**：微信公众号发布
- **模式**：浏览器模式 / API模式
- **用途**：自动发布到公众号

### 5. baoyu-post-to-x
- **功能**：X/Twitter发布
- **类型**：普通推文 / X Articles
- **用途**：自动发布到X

---

## 🚀 快速开始

## ⚙️ 配置说明

### 1. 浏览器模式（推荐）
无需配置，直接使用Chrome自动化

**优点**：
- 无需API密钥
- 开箱即用
- 支持所有功能

**使用方法**：
- 首次使用时会打开Chrome
- 手动登录账号
- 后续自动化操作

### 2. API模式（可选）
需要配置微信公众号API

**配置文件**：`~/.baoyu-skills/.env`
```bash
WECHAT_APP_ID=your_app_id
WECHAT_APP_SECRET=your_app_secret
```

**优点**：
- 更稳定
- 更快速
- 适合批量操作

## ⚠️ 注意事项

### 1. Chrome依赖
- 确保Chrome已安装
- 首次使用需要手动登录
- 保持Chrome版本更新

### 2. 网络要求
- 采集功能需要稳定网络
- 部分网站可能有反爬虫
- 建议使用等待模式处理登录页面

### 3. 账号安全
- 使用专用账号进行自动化
- 定期检查发布内容
- 遵守平台规则

### 4. 内容审核
- 发布前人工审核
- 确保符合平台规范
- 注意版权和原创性

### 5. 频率限制
- 避免频繁发布
- 设置发布间隔
- 防止账号异常

## 🐛 故障排查

### Q1: Chrome无法启动
```
解决方案：
1. 检查Chrome是否安装
2. 更新到最新版本
3. 检查系统权限
```

### Q2: 网页采集失败
```
解决方案：
1. 检查网络连接
2. 尝试等待模式
3. 手动登录后再采集
4. 检查网站是否有反爬虫
```

### Q3: 格式化异常
```
解决方案：
1. 检查Markdown语法
2. 手动调整特殊格式
3. 使用预设模板
```

### Q4: 发布失败
```
解决方案：
1. 检查账号登录状态
2. 确认内容符合规范
3. 检查图片大小和格式
4. 重新登录账号
```

### Q5: 依赖安装失败
```
解决方案：
1. 检查Node.js版本
2. 使用npm install重新安装
3. 清除缓存：npm cache clean --force
4. 使用国内镜像：npm config set registry https://registry.npmmirror.com
```
