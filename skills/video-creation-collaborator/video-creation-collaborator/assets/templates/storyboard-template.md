## 填写指南

### 镜头序号
- 格式: L01, L02, L03...
- 顺序: 从1开始,依次递增

### 时长
- 单镜头时长: 2-5秒
- 总时长: 严格匹配视频时长(30秒/45秒/60秒)
- 时长分配: 根据内容重要性分配,关键镜头可稍长

### 构图
- 可选项: 特写/中景/全景/移动镜头
- 选择原则: 根据内容需要选择

### 场景描述
- 内容: 描述场景环境、光影、色调
- 要求: 具体明确,避免模糊表述
- 示例: "城市公园,晴朗,自然光,冷蓝色调"

### 画面动作
- 内容: 描述人物/商品的动作
- 要求: 具体明确,明确人体肢体姿态
- 示例: "青年跑步,手臂自然摆动,手腕佩戴智能手表,手指舒展"
- 注意: 避免模糊表述如"青年运动",应明确"青年跑步"

### 文案
- 内容: 分镜适配文案
- 要求: 每句4-8字,与镜头内容匹配
- 示例: "腕间掌控", "智能连接"

### 音效备注
- 内容: 背景音乐/场景音效
- 要求: 标注音效类型、时长、音量
- 示例: "背景音乐(轻快电子音),脚步声,时长3秒,音量30%"

### 转场偏好
- 可选项: 淡入淡出/切镜/柔焦渐变/胶片叠化
- 选择原则: 根据内容需要选择
- 时长: 0.3-0.5秒

## 质量检查清单

### 格式检查
- [ ] 镜头序号连续(L01, L02, L03...)
- [ ] 总时长严格匹配(30秒/45秒/60秒)
- [ ] 单镜头时长2-5秒
- [ ] 构图标注完整(特写/中景/全景)
- [ ] 场景描述具体明确
- [ ] 画面动作具体明确,明确肢体姿态
- [ ] 文案每句4-8字
- [ ] 音效备注完整(类型/时长/音量)
- [ ] 转场偏好标注完整

### 内容检查
- [ ] 镜头数量符合要求(30秒8-10个,45秒12-15个,60秒15-18个)
- [ ] 故事线逻辑连贯
- [ ] 场景设定符合目标受众偏好
- [ ] 动作设计贴合现实,无违背生理结构的场景
- [ ] 文案风格与视频调性一致
- [ ] 文案贴合商品卖点

### 质量检查
- [ ] 无模糊表述,规避生图失真风险
- [ ] 人体肢体姿态描述清晰
- [ ] 商品细节要求明确
- [ ] 转场时长标注(0.3-0.5秒)

---

## 注意事项

1. **肢体姿态**: 画面动作描述必须明确人体肢体姿态,避免模糊表述导致的生图失真
2. **商品细节**: 商品细节要求必须明确,确保生图时精准还原
3. **转场标注**: 相邻场景必须标注转场方式,避免视频合成时出现突兀切换
4. **时长分配**: 严格匹配视频总时长,单镜头时长2-5秒,确保节奏合理
5. **风格统一**: 场景描述、文案、音效必须与视频风格统一

**备注**: 所有填写内容必须符合【质检标准与负面规避清单】,避免出现瑕疵。
