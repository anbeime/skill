本参考文档提供不同情绪、场景下的音乐风格搭配建议，帮助生成适合古诗词意境的纯音乐配乐蓝图。

## 情绪与音乐风格

### 宁静与平和
**适用场景**：田园诗、隐逸诗、山水诗的静谧场景
**风格标签**：Ambient, Calm, Peaceful, Meditative
**乐器建议**：Guzheng（古筝）、Pipa（琵琶）、Dizi（笛子）、Harp（竖琴）、Atmospheric Pads
**节奏**：Slow, Rubato（自由节奏）
**示例标签组合**：
```
China-Chic, Ambient, Calm, Peaceful, Meditative, Guzheng, Flute, Harp, Atmospheric Pads, Slow Build, Rubato
```

### 忧伤与思念
**适用场景**：离别诗、思乡诗、怀古诗
**风格标签**：Melancholic, Sad, Nostalgic, Sorrowful
**乐器建议**：Erhu（二胡）、Suona（唢呐，慎用）、Strings（弦乐）、Piano（钢琴）
**节奏**：Slow, Rubato, Gradual Build
**示例标签组合**：
```
China-Chic, Cinematic, Melancholic, Nostalgic, Sorrowful, Erhu, Strings, Piano, Atmospheric Pads, Slow, Rubato
```

### 激昂与豪迈
**适用场景**：边塞诗、咏史诗、豪放词
**风格标签**：Epic, Powerful, Majestic, Heroic
**乐器建议**：Suona（唢呐）、Gongs（锣）、Drums（鼓）、Orchestral Brass（铜管乐）、Taiko Drum
**节奏**：Mid-tempo, Powerful Beats, 4/4 Beat
**示例标签组合**：
```
China-Chic, Epic, Cinematic, Powerful, Majestic, Heroic, Suona, Orchestral Brass, Taiko Drum, Strings, Mid-tempo, 4/4 Beat
```

### 欢快与明朗
**适用场景**：田园诗、春景诗、节日诗
**风格标签**：Uplifting, Joyful, Cheerful, Bright
**乐器建议**：Pipa（琵琶）、Dizi（笛子）、Percussion（轻打击乐）、Flute（长笛）
**节奏**：Mid-tempo, Upbeat, Playful Rhythm
**示例标签组合**：
```
China-Chic, Uplifting, Joyful, Cheerful, Bright, Pipa, Dizi, Percussion, Strings, Mid-tempo, Playful
```

### 神秘与空灵
**适用场景**：道家诗、佛教诗、山水诗的超然场景
**风格标签**：Mysterious, Ethereal, Space, Dreamy
**乐器建议**：Guzheng（古筝）、Gong（锣）、Chimes（钟）、Synth Pads（合成器铺底）、Wind Chimes（风铃）
**节奏**：Slow, Rubato, Minimal Rhythm
**示例标签组合**：
```
China-Chic, Ambient, Mysterious, Ethereal, Space, Dreamy, Guzheng, Gong, Chimes, Wind Chimes, Synth Pads, Slow, Rubato
```

### 温馨与温暖
**适用场景**：友情诗、家庭诗、归家诗
**风格标签**：Warm, Cozy, Gentle, Tender
**乐器建议**：Pipa（琵琶）、Erhu（二胡，轻柔）、Strings（弦乐）、Flute（长笛）
**节奏**：Slow to Mid-tempo, Gentle Flow
**示例标签组合**：
```
China-Chic, Warm, Cozy, Gentle, Tender, Pipa, Erhu, Strings, Flute, Atmospheric Pads, Slow Build, Gentle Flow
```

## 中国乐器特性

### Guzheng（古筝）
- **音色**：清脆悦耳，音域宽广，擅长表现流水、风声等自然音
- **情感**：适合表现宁静、思念、豪放等多种情绪
- **技法**：揉弦、滑音、颤音、刮奏
- **搭配建议**：可与笛子、二胡、弦乐完美融合

### Pipa（琵琶）
- **音色**：清脆明亮，表现力丰富，擅长表现叙事、战斗场景
- **情感**：适合表现欢快、激昂、紧张等情绪
- **技法**：轮指、扫弦、滚奏、泛音
- **搭配建议**：与笛子、鼓、弦乐组合效果佳

### Erhu（二胡）
- **音色**：深情醇厚，如人声般富有表现力
- **情感**：擅长表现忧伤、思念、悲痛等深沉情绪
- **技法**：揉弦、滑音、颤音、拨弦
- **搭配建议**：与古筝、弦乐、钢琴配合出色

### Dizi（笛子）
- **音色**：清脆悠扬，如鸟鸣般清亮
- **情感**：适合表现欢快、明朗、空灵等情绪
- **技法**：颤音、滑音、花舌、泛音
- **搭配建议**：与古筝、琵琶、弦乐和谐共处

### Suona（唢呐）
- **音色**：高亢嘹亮，穿透力强
- **情感**：适合表现激昂、热烈、悲壮等强烈情绪
- **技法**：滑音、颤音、吐音
- **搭配建议**：与鼓、锣、铜管乐组合震撼

### Guqin（古琴）
- **音色**：深沉古朴，如金石之声
- **情感**：适合表现超然、哲理、内省等意境
- **技法**：滑音、揉弦、泛音、滚拂
- **搭配建议**：单独演奏或与极简乐器配合

## 音乐结构设计

### 单段式（A-A-A）
**适用场景**：简单场景，情绪单一
**结构**：Intro → Verse → Outro
**特点**：重复同一主题，适合表现宁静、冥想

### 二段式（A-B-A-B）
**适用场景**：情绪对比，叙事性强
**结构**：Intro → Verse → Chorus → Verse → Chorus → Outro
**特点**：主歌与副歌对比，适合表现情绪递进

### 三段式（A-B-A）
**适用场景**：情感转折，戏剧性强
**结构**：Intro → Verse → Chorus → Bridge → Chorus → Outro
**特点**：桥段提供转折，适合表现复杂情感

### 变奏式
**适用场景**：主题发展，层次丰富
**结构**：Intro → Theme → Variation 1 → Variation 2 → Theme → Outro
**特点**：同一主题的多次变奏，适合表现细腻变化

### 回旋式
**适用场景**：主题反复，场景转换
**结构**：Intro → A → B → A → C → A → Outro
**特点**：主歌反复出现，适合表现场景切换

## 注意事项
1. **乐器搭配**：注意乐器之间的和谐与平衡
2. **情绪递进**：设计合理的情绪曲线，避免单调
3. **节奏变化**：在适当的地方调整节奏，增加音乐层次
4. **音效运用**：谨慎使用环境音效，增强画面感
5. **风格统一**：保持整体风格的一致性，避免突兀变化
6. **时长控制**：根据场景需求控制音乐长度
7. **文化尊重**：正确使用中国乐器，避免文化误用
