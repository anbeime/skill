# 3D纸艺弹出书插画生成

## 任务目标
- 本 Skill 用于：生成具有3D立体纸艺效果的弹出书风格插画
- 能力包含：结合纸质纹理与数字科技元素，创造温暖而富有科技感的视觉作品
- 触发条件：用户需要生成具有纸艺质感、立体层次、特定色调（薄荷绿/天蓝/珊瑚橙/奶油黄）的插画时

## 核心设计要素

### 色彩方案
- 主色调：薄荷绿（清新轻盈）
- 辅助色：天蓝色（科技感）、珊瑚橙（活力）、奶油黄（温暖）
- 整体氛围：明亮温暖，避免深沉或冷峻色调

### 视觉特征
- 纸质纹理质感：纸张表面触感、折痕、纹理细节
- 立体层次：多层弹出结构，前后景深明显
- 书本场景：打开的书本，书页边缘可见，营造真实翻书效果
- 数字科技元素：霓虹光效、科技图标、抽象数据流，与纸质风格融合

### 效果细节
- 柔和霓虹光效：点缀式光效，不破坏纸质质感
- 温暖科技光照：暖色调光照配合科技元素
- 立体阴影：增强层次感和真实感
- 细节丰富：包含精美的装饰元素和关键信息展示

### 风格定位
- 整体风格：圆润可爱但专业精致
- 视觉递进：从简单到复杂的视觉层次
- 质感结合：纸质手工质感与数字科技感的平衡

## 操作步骤

### 步骤1：理解场景需求
- 确认主题内容：科技、教育、品牌、产品等
- 明确关键信息：需要突出的文字、图标、元素
- 确定复杂程度：单层弹出vs多层立体结构

### 步骤2：构建图像生成提示词
根据核心设计要素，构建包含以下元素的提示词：

**基础结构：**
```
3D纸艺弹出书风格插画，立体层次感，打开的书本场景，书页边缘清晰可见
```

**色彩添加：**
```
明亮温暖色调，薄荷绿、天蓝色、珊瑚橙、奶油黄配色
```

**质感描述：**
```
纸质纹理质感，折痕细节，手工制作感，纸张表面触感
```

**立体效果：**
```
多层弹出结构，前后景深，立体阴影，真实立体感
```

**科技元素（如需要）：**
```
柔和霓虹光效点缀，数字科技元素，科技图标，数据流线条，温暖科技光照
```

**风格特征：**
```
圆润可爱但专业精致，细节丰富，精美装饰
```

### 步骤3：生成并验证
- 生成图像后检查：
  - 立体层次是否明显
  - 色调是否符合明亮温暖要求
  - 纸质质感是否真实
  - 科技元素是否与纸质风格融合自然
  - 细节是否丰富但不杂乱

3D纸艺弹出书风格插画，打开的书本中央弹出AI大脑图案，周围是神经网络线条，纸质纹理质感，立体层次明显，薄荷绿和天蓝色为主色调，柔和霓虹光效点缀，温暖科技光照，书页边缘可见，圆润可爱但专业精致

3D纸艺弹出书风格插画，打开的书本弹出多层知识树，树上有书本、灯泡、齿轮等元素，珊瑚橙和奶油黄为主色调，纸质折痕细节丰富，立体阴影增强层次感，书页边缘清晰，从简单到复杂的视觉递进，温暖明亮氛围

## 注意事项
- 保持纸质质感的主导地位，科技元素为辅助点缀
- 确保立体层次清晰，避免扁平化效果
- 色调始终保持明亮温暖，避免冷峻或暗沉
- 细节丰富但要有视觉重心，避免过于杂乱
- 书页边缘和翻书效果是关键识别特征，需清晰呈现

## 适用场景
- 科技产品宣传：展示AI、云服务、数据等产品特性
- 教育内容呈现：知识图谱、学习路径、课程大纲
- 品牌故事讲述：品牌理念、发展历程、核心价值
- 信息图表设计：数据可视化、流程说明、层级关系
- 活动视觉物料：海报、邀请函、宣传册封面
