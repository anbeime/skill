# 股票代码格式参考

## A股代码格式

### 深交所（深圳证券交易所）
- 格式：`6位数字` 或 `6位数字.SZ`
- 示例：
  - `000001` 或 `000001.SZ` - 平安银行
  - `000002` 或 `000002.SZ` - 万科A
  - `300001` 或 `300001.SZ` - 特锐德（创业板）

### 上交所（上海证券交易所）
- 格式：`6位数字` 或 `sh6位数字`
- 示例：
  - `600000` 或 `sh600000` - 浦发银行
  - `600036` 或 `sh600036` - 招商银行
  - `688001` 或 `sh688001` - 中微公司（科创板）

### 北交所（北京证券交易所）
- 格式：`8或4开头的6位数字`
- 示例：
  - `830799` - 艾融软件

## 港股代码格式
- 格式：`5位数字.HK` 或 `5位数字`
- 示例：
  - `00700.HK` 或 `00700` - 腾讯控股
  - `00388.HK` 或 `00388` - 港交所
  - `09988.HK` 或 `09988` - 阿里巴巴

## 美股代码格式
- 格式：`股票代码`（字母）
- 示例：
  - `AAPL` - 苹果公司
  - `TSLA` - 特斯拉
  - `MSFT` - 微软

## 注意事项
- A股代码建议使用简化格式（6位数字），脚本会自动识别交易所
- 港股建议加上`.HK`后缀
- 美股直接使用字母代码
- 如果数据获取失败，尝试不同的代码格式
