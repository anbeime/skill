# 质量标准规范

## 概览
本规范定义了商品视频创作的质量标准，包括图片质量、视频质量和音画同步等方面的要求。

## 图片质量标准

### 分辨率要求
- **最低要求**：1920x1080（1080P）
- **推荐分辨率**：1920x1080（横屏）、1080x1920（竖屏）
- **说明**：确保图片清晰度，支持后续视频合成的高质量输出

### 人体结构标准
- **肢体数量**：仅允许1-2只正常手臂，无多余肢体
- **肢体比例**：手腕-手肘-肩膀距离比例应在0.3-0.8之间
- **手部状态**：无畸形、无错位、无重影、无多余手指
- **肢体位置**：关键点应在画面内（0 < x < 1, 0 < y < 1）

### 画质要求
- **模糊度**：拉普拉斯方差应≥100（数值越大越清晰）
- **噪点**：无明显噪点，画面细腻
- **色彩**：色彩饱和度适中，无过曝或欠曝
- **对比度**：对比度合理，细节清晰可见

### 商品完整性标准
- **可见性**：商品必须在画面中清晰可见
- **细节**：商品细节（纹理、按键、屏幕显示）应清晰
- **位置**：商品位置合理，不遮挡主体
- **变形**：无变形、无扭曲、无模糊

### 场景质量标准
- **元素真实**：场景元素（人物、环境、道具）真实可信
- **光影均匀**：光线均匀，无过曝或阴影死角
- **风格统一**：所有图片风格一致，色调统一
- **无杂质**：无多余装饰元素，画面干净整洁

## 视频质量标准

### 分辨率标准
- **横屏**：1920x1080（16:9）
- **竖屏**：1080x1920（9:16）
- **说明**：根据发布平台选择合适的格式

### 帧率标准
- **推荐**：25 fps
- **可选**：24 fps（电影感）、30 fps（更流畅）
- **说明**：帧率稳定，无卡顿

### 码率标准
- **1080P/25fps**：8-15 Mbps
- **说明**：码率越高画质越好，但文件也越大

### 转场标准
- **转场时长**：0.3-0.8秒
- **转场类型**：
  - 智能硬件类：淡入淡出+科技光效
  - 美妆护肤类：柔焦渐变
  - 家居家电类：胶片感叠化
- **转场质量**：无画面撕裂、无色彩断层、过渡自然流畅

### 画面质量
- **清晰度**：画面清晰，无马赛克
- **色彩**：色彩准确，无色差
- **稳定性**：动作流畅，无卡顿、无跳屏
- **帧完整性**：无帧丢失、无重复帧

## 音频质量标准

### 音频格式
- **编码格式**：AAC
- **采样率**：44.1 kHz或48 kHz
- **比特率**：128-192 kbps

### 音频内容
- **背景音乐**：
  - 智能硬件类：轻快有节奏的电子音
  - 美妆护肤类：轻柔的钢琴曲
  - 家居家电类：慵懒的爵士乐
- **音效**：
  - 环境音：真实场景音效（如面膜撕开、咖啡机萃取）
  - 转场音效：配合转场效果的音效
  - 商品音效：点击声、机械声等

### 音量平衡
- **背景音乐**：比人声/音效低15-20 dB
- **音效**：清晰可闻，不刺耳
- **音量稳定**：无突兀的音量变化

## 音画同步标准

### 同步精度
- **音频偏移**：≤0.1秒
- **字幕同步**：字幕与画面精准对应
- **转场同步**：转场与音频节奏匹配

### 字幕标准
- **位置**：画面下方1/3处，避免遮挡商品主体
- **字体大小**：
  - 1920x1080：48-72像素
  - 1080x1920：36-54像素
- **颜色**：白色（#FFFFFF）
- **背景**：半透明黑色条（rgba(0,0,0,0.5)）
- **显示时间**：2-4秒

## 质量校验流程

### 图片校验
1. **画质检测**：使用拉普拉斯方差检测模糊度
2. **肢体检测**：使用mediapipe检测肢体异常
3. **商品检测**：检测商品完整性和清晰度
4. **合格标准**：所有检测项均通过

### 视频校验
1. **分辨率检查**：确保视频分辨率符合要求
2. **帧率检查**：确保帧率稳定
3. **完整性检查**：无帧丢失、无跳屏
4. **音画同步**：检查音频偏移是否在允许范围内

### 最终校验
1. **视觉检查**：人工检查画面质量
2. **听觉检查**：人工检查音频质量
3. **字幕检查**：人工检查字幕清晰度和位置
4. **整体检查**：整体观看体验评估

## 质量问题处理

### 图片质量问题
- **模糊**：重新生成图片，提高清晰度
- **肢体异常**：重新生成图片，调整人体姿势
- **商品变形**：重新生成图片，修正商品细节
- **处理流程**：标记问题 → 反馈给分镜画师 → 重新生成 → 再次校验

### 视频质量问题
- **卡顿**：降低码率或调整转场时长
- **色彩失真**：调整视频编码参数
- **音画不同步**：调整音频起始时间
- **处理流程**：标记问题 → 调整参数 → 重新合成 → 再次校验

## 质量标准对比

| 质量等级 | 分辨率 | 帧率 | 码率 | 音画同步 | 适用场景 |
|---------|-------|------|------|---------|---------|
| 标准 | 1920x1080 | 25 fps | 8 Mbps | ≤0.1秒 | 电商平台、社交媒体 |
| 高级 | 1920x1080 | 25 fps | 12 Mbps | ≤0.05秒 | 品牌宣传、官网 |
| 专业 | 3840x2160 | 30 fps | 20 Mbps | ≤0.02秒 | 电视广告、高端品牌 |

## 常见质量问题

### 图片常见问题
1. **多余肢体**：检测到超过2只手臂
2. **手部畸形**：手腕-手肘-肩膀比例异常
3. **画面模糊**：拉普拉斯方差<100
4. **商品变形**：商品边缘检测不清晰
5. **色彩失真**：平均亮度<30或>225

### 视频常见问题
1. **卡顿**：码率过低或转场过长
2. **色彩失真**：编码参数不当
3. **音画不同步**：音频偏移>0.1秒
4. **字幕不清**：字体大小或颜色不当
5. **转场撕裂**：转场时长过长或参数不当

## 质量提升建议

### 图片质量提升
1. 提高生成图片的分辨率和质量参数
2. 在提示词中强调质量约束和负面提示词
3. 生成后进行质量校验，不合格图片重新生成
4. 使用专业的图片处理工具进行后期优化

### 视频质量提升
1. 使用高码率（8-15 Mbps）
2. 选择合适的帧率（25 fps）
3. 优化转场效果，确保流畅自然
4. 调整音画同步，确保精度≤0.1秒
5. 使用专业的视频编码参数

### 音频质量提升
1. 选择高质量的背景音乐
2. 精心设计音效方案
3. 调整音量平衡，确保背景音乐不干扰字幕
4. 使用高质量的音频编码（AAC，128-192 kbps）
