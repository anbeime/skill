## 案例1：智能硬件类（30秒，智能手环）

### 提问词
```
帮我做一条30秒的智能手环宣传视频，重点突出心率监测、7天续航和防水防尘三大卖点，目标受众25-40岁上班族。风格为简约科技风，画面包含职场通勤（办公室、地铁）和运动健身（慢跑、瑜伽）双场景，仅展示正常人体肢体（手臂无多余肢体、无畸形、无模糊重影），手环细节清晰可见（表带纹理、屏幕显示、按键布局），画面分辨率不低于1080P，无噪点、无虚化断层。背景音乐选轻快有节奏的电子音，字幕简单有力突出卖点。转场用淡入淡出+科技光效（转场流畅无卡顿、无画面撕裂），音频与画面精准同步，字幕叠加位置合理（不遮挡商品主体），最后输出完整MP4视频，成片无帧丢失、无色彩失真。
```

### 前置准备：安装依赖（**必须执行**）
```bash
pip install moviepy>=1.0.3 pillow>=10.0.0 opencv-python>=4.8.0 mediapipe>=0.10.0 numpy>=1.24.0
```

### 执行流程

#### 步骤1：文案创作师
- 核心卖点：心率监测、7天续航、防水防尘
- 目标受众：25-40岁上班族
- 输出文案：
  - 标题："智能生活，腕间掌控"
  - 核心宣传语："实时监测，健康常伴"
  - 卖点详情：
    - 心率监测，实时精准
    - 超长续航，无忧使用
    - 防水防尘，无惧挑战

#### 步骤2：故事策划师
- 故事主题：双场景展示
- 场景设计：
  - 职场通勤场景：办公室工作、地铁通勤
  - 运动健身场景：健身房慢跑、瑜伽练习
- 故事线：工作→通勤→运动→生活

#### 步骤3：脚本创作师
- 镜头数量：8个
- 总时长：30秒
- 转场方式：淡入淡出+科技光效

#### 步骤4：分镜导演
- 质量约束提示词（内嵌）：
  1. 人体结构精准：仅展示正常成年人单侧手臂佩戴手环，无多余肢体、无重影、无畸形
  2. **角色描述一致**：明确主角为25-40岁上班族（如：男性，短发，穿白衬衫/运动服），8张图片中同一人物的服装、发型、外貌、风格必须完全一致
  3. 商品细节清晰：手环表带纹理、屏幕显示、按键位置精准还原
  4. 场景质感要求：职场场景元素真实，色彩饱和度适中，光线均匀，1080P
  5. 风格统一约束：简约科技风，色调以冷白、浅灰、科技蓝为主
- 负面规避提示词（内嵌）：
  - 禁止多余肢体、手部畸形、手环变形、画面重影、噪点、虚化断层、色彩失真
  - **禁止人物不一致**：同一人物在不同镜头间服装、发型、外貌、风格发生突变

#### 步骤5：分镜画师
- 生成8张场景图片
- **角色一致性**：确保8张图片中同一人物的服装、发型、外貌、风格完全一致（如：始终穿同一件白衬衫或同一套运动服）
- 应用质量约束和负面提示词
- 画面风格：简约科技风

#### 步骤6：质量校验
```bash
python scripts/image_quality_checker.py \
  --images ./scene_*.jpg \
  --output ./quality_report.json
```
- 检测肢体异常
- 检测画质问题
- 检测手环细节清晰度
- 不合格图片标记重绘

#### 步骤7：字幕师
- 字幕风格：简单有力
- 字幕位置：画面下方1/3处
- 字幕内容：
  - 心率监测，实时精准
  - 全面数据，一目了然
  - 运动追踪，精准记录
  - 超长续航，无忧使用
  - 智能提醒，从不遗漏
  - 防水防尘，无惧挑战

#### 步骤8：音效师
- 背景音乐：轻快有节奏的电子音
- 音效：转场音效、点击声
- 音量平衡：背景音乐比音效低15-20 dB

#### 步骤9：视频工程师
```bash
python scripts/video_composer.py \
  --images ./scene_*.jpg \
  --subtitles ./subtitles.json \
  --audio ./bgm.mp3 \
  --output ./final_video.mp4 \
  --fps 25 \
  --width 1920 \
  --height 1080 \
  --bitrate 8M
```

### 能力展示点
- 文案创作师提炼手环核心卖点
- 故事策划师搭建职场、运动双场景剧情
- 分镜导演规划科技风画面及转场效果
- 分镜画师生成对应场景图（质量约束）
- 质量校验检测肢体异常、确保手环细节清晰
- 音效师匹配合适的电子音BGM
- 视频工程师处理分辨率调整、转场叠加、音画同步

## 案例2：美妆护肤类（60秒，玻尿酸面膜）

### 提问词
```
我有一款玻尿酸面膜，主打的是温和补水，敏感肌也能用，想做一条60秒的情感向宣传视频，目标受众25-35岁熬夜党女性。核心围绕"熬夜急救"场景展开，故事线聚焦"熬夜后皮肤干燥泛红→敷面膜舒缓→敷完水润透亮"的前后对比，逻辑连贯贴合补水卖点。风格定温柔治愈风，画面色调暖调（米白+浅粉为主），拆分15-18个镜头（单镜头时长3-4秒，总时长严格适配60秒）。角色设定统一为"28岁女性，妆容淡雅，手部动作自然舒展"，全程保持角色形象一致。字幕需有代入感、文本精炼，背景音乐用轻柔钢琴曲，搭配面膜撕开、敷贴、轻拍面部的真实场景音效（音量适中，不盖过背景音）。转场统一用柔焦渐变，适配抖音、视频号竖屏9:16比例，画质1080P无噪点。视频合成调用COZE平台视频大模型API实现，无需依赖本地包，支持图片序列、字幕、音频同步合成，规避合成死循环，确保转场流畅、音画同步，无画面撕裂或帧丢失。
```

### 前置准备：安装依赖（**必须执行**）

  - 禁止角色形象不一致、手部动作畸形、面膜变形、面部皮肤状态突变

- 生成15-18张场景图片
- **角色一致性**：确保角色全程一致（发型、服装、妆容无变化）
- 重点：皮肤前后对比（从干燥泛红到水润透亮）
- 画面风格：温柔治愈风，暖色调

#### 步骤6：质量校验
```bash
python scripts/image_quality_checker.py \
  --images ./scene_*.jpg \
  --output ./quality_report.json
```

#### 步骤7：字幕师
- 字幕风格：有代入感，情感化，文本精炼

- 字幕内容：
  - 熬夜后的疲惫
  - 温和补水，敏感肌友好
  - 敷上肌肤，即刻吸收
  - 10分钟见证改变
  - 水润透亮，焕发光彩
  - 紧致提亮，重现青春

#### 步骤8：音效师
- 背景音乐：轻柔的钢琴曲
- 环境音：面膜撕开声、敷上脸声音、轻拍面部声音
- 音量平衡：背景音乐比环境音低15-20 dB

#### 步骤9：视频工程师（API调用方案）
```bash
python scripts/video_composer.py \
  --api-mode \
  --images ./scene_*.jpg \
  --subtitles ./subtitles.json \
  --audio ./bgm.mp3 \
  --output ./final_video.mp4 \
  --resolution 1080P \
  --aspect-ratio 9:16 \
  --fps 25 \
  --transition soft-fade \
  --timeout 10 \
  --retry 3
```
- 合成逻辑：调用COZE平台视频大模型API
- 死循环规避：超时重试机制（超时10秒，重试≤3次）
- API参数配置：分辨率1080P、帧率25fps、比例9:16、转场柔焦渐变
- 音画同步校验：音频偏移≤0.1秒

- 故事策划师的情感剧情设计能力
