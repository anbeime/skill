# 玻尿酸面膜视频专属创作指令（COZE API适配版）

## 商品信息
- **商品名称**：玻尿酸面膜
- **核心卖点**：温和补水、敏感肌适用、熬夜急救
- **目标受众**：25-35岁熬夜党女性
- **视频时长**：60秒

## 故事线与场景设计

### 核心故事
围绕"熬夜急救"场景展开，故事线聚焦"熬夜后皮肤干燥泛红→敷面膜舒缓→敷完水润透亮"的前后对比，逻辑连贯贴合补水卖点

### 场景拆分
- 深夜加班，皮肤干燥泛红（镜头1-3）
- 敷上面膜，温和吸收（镜头4-8）
- 摘下面膜，水润透亮（镜头9-12）
- 清晨醒来，自信面对（镜头13-18）

## 核心创作要求

### 1. 风格要求
- **整体风格**：温柔治愈风
- **画面色调**：统一为暖调（米白+浅粉为主），无色调偏移

### 2. 镜头要求
- **镜头数量**：拆分15-18个镜头
- **单镜头时长**：3-4秒
- **总时长**：严格60秒，无偏差
- **转场效果**：柔焦渐变（0.3-0.5秒）

### 3. 角色要求
- **角色数量**：全程仅展示1个28岁女性
- **角色特征**：
  - 鹅蛋脸
  - 淡雅裸妆
  - 低马尾
  - 米白色家居服
  - 手部皮肤白皙细腻、手指舒展自然
- **角色一致性**：所有镜头角色形象完全一致，面部皮肤状态随剧情自然过渡（熬夜泛红→敷膜舒缓→水润透亮）

### 4. 字幕要求
- **字幕风格**：有代入感、文本精炼（4-8字/句）
- **字幕输出格式**：**COZE视频大模型API可直接解析的参数包**（含显示时长、位置、格式、同步节点）
- **字幕位置**：竖屏9:16画面底部居中，距底边30px，距左右边距20px
- **字幕格式**：
  - 字体：柔和无衬线体
  - 字号：28号（1080P竖屏适配）
  - 颜色：浅杏色/奶白色（与暖调画面对比且不刺眼）
  - 描边：浅粉色1px（增加辨识度）
- **核心要求**：绝对不遮挡女性面部、面膜主体

### 5. 音频要求
- **背景音乐**：60秒轻柔钢琴曲（无歌词，治愈舒缓）
- **场景音效**：
  - 撕面膜声
  - 敷面膜声
  - 轻拍面部声
- **音量配置**：
  - 背景音乐音量：60%
  - 场景音效音量：30%-40%（不盖过背景音乐）
- **同步要求**：音效与字幕、画面动作精准同步
- **音频输出格式**：**COZE API适配的音效素材包**（MP3格式）

### 6. 合成要求
- **合成方式**：**唯一调用COZE平台视频大模型API完成全流程合成**
- **禁止事项**：
  - 无任何本地依赖包调用行为
  - 无本地脚本合成
- **视频参数**：
  - 分辨率：1080P
  - 比例：竖屏9:16
  - 帧率：25fps
  - 转场：柔焦渐变（0.3-0.5秒）
  - 输出格式：MP4
- **功能开启**：
  - 高清降噪
  - 音画精准同步（音频偏移≤0.1秒）
- **核心要求**：确保转场流畅、音画同步、无画面撕裂/帧丢失/噪点

### 7. 商品细节要求
- 面膜包装纹理精准还原
- 膜布质感清晰可见
- 精华液光泽自然真实
- 与面部贴合自然
- 无变形/悬浮/模糊

## COZE API适配清单

### 图片素材
- 格式：JPG/PNG（COZE API优选格式）
- 分辨率：1080P
- 比例：竖屏9:16
- 数量：15-18张
- 命名规范：镜头序号_场景名称（如01_熬夜泛红）

### 字幕参数包
- 格式：COZE API可直接解析的列表格式
- 包含：镜头序号、字幕文本、显示时长（开始时间/结束时间）、格式参数（字体/字号/颜色/描边）、位置参数、同步参数
- 示例：
  ```
  镜头01｜文本：熬夜脸泛红，干燥紧绷｜显示时长：0-3秒｜字体：柔和无衬线体｜字号：28｜颜色：奶白色｜描边：浅粉色1px｜位置：底部居中（距底30px）｜同步节点：0秒熬夜面部特写出现时
  ```

### 音效素材包
- 格式：MP3（COZE API优选格式）
- 背景音乐：60秒轻柔钢琴曲，音量60%
- 场景音效：按镜头序号匹配，标注每个音效的开始时间/结束时间/音量参数（30%-40%）
- 示例：
  ```
  镜头03｜音效：撕面膜声｜时长：9-10秒｜音量：35%｜同步节点：字幕"轻撕膜布，满满精华"显示时
  ```

### API合成参数
- 分辨率：1080P
- 视频比例：9:16
- 总时长：60秒
- 帧率：25fps
- 输出格式：MP4
- 转场：柔焦渐变（0.3-0.5秒）
- 音画同步：开启API自带精准同步功能
- 高清降噪：开启API自带高清降噪功能

## 执行流程

### 步骤1：文案创作师
- 生成核心Slogan（8-12字）
- 生成分镜适配文案（15-18句，每句4-8字）

### 步骤2：故事策划师
- 设计"熬夜急救"场景故事线
- 拆分15-18个镜头（单镜头3-4秒）
- 明确每个镜头的核心场景、动作、卖点关联点

### 步骤3：脚本创作师
- 编写COZE API适配的标准化分镜脚本
- 标注转场为"柔焦渐变（0.3-0.5秒）"
- **脚本中无任何本地依赖包相关表述**

### 步骤4：分镜导演
- 明确28岁女性统一角色规范（鹅蛋脸、淡雅裸妆、低马尾、米白色家居服）
- 标注暖调（米白+浅粉）画面色调
- 明确人体肢体规范（无多余肢体、无重影、无畸形）
- **所有输出图片需为COZE API直传格式**

### 步骤5：分镜画师
- 生成15-18张COZE API可上传的1080P竖屏9:16图片（JPG/PNG格式）
- **全15-18张图片中角色形象、服装、妆容完全统一**
- 面部皮肤状态随剧情自然过渡（熬夜泛红→敷膜舒缓→水润透亮）
- **生成后直接进入COZE API素材池，无本地校验/处理步骤**

### 步骤6：字幕师
- 输出**COZE API字幕参数包**（按镜头序号排序，可直接复制上传API）
- 字幕格式：柔和无衬线体、字号28、颜色奶白色、描边浅粉色1px
- 字幕位置：画面底部居中（距底30px，距左右边距20px）
- 标注字幕与音效的时间同步节点

### 步骤7：音效师
- 输出**COZE API音效素材包**（背景音乐60秒轻柔钢琴曲，场景音效按镜头匹配）
- 背景音乐音量60%，场景音效音量30%-40%
- 标注每个音效的开始时间/结束时间/音量参数

### 步骤8：视频工程师（唯一COZE API合成）
- 素材预检：校验图片、字幕参数包、音效素材包是否符合API要求
- **不启动任何本地包校验/处理**
- API参数配置：1080P、9:16、60秒、25fps、柔焦渐变转场、音画同步、高清降噪
- 素材上传：将分镜图片、字幕参数包、音效素材包**直接上传至COZE API素材池**
- 合成执行：启动API合成，开启**合成状态实时回调**，若异常触发**API重试机制**（≤3次，超时10秒）
- 成品输出：直接从COZE API获取MP4成品视频

### 步骤9：质检智能体
- 重点校验：是否存在本地依赖包相关操作
- 重点校验：所有输出物是否适配COZE API格式
- 重大瑕疵（依赖包违规、素材不适配API、角色不一致、字幕遮挡）直接拦截输出

### 步骤10：数据反馈智能体
- 统计本地依赖包违规次数
- 统计API适配不合格率
- 统计字幕参数包适配合格率
- 输出迭代优化建议

## 注意事项

1. **API合成强制**：唯一调用COZE平台视频大模型API完成视频合成，无本地依赖包调用
2. **流程阻断规则**：若出现本地依赖包调用行为，立即终止并切换至纯COZE API合成流程
3. **角色一致性**：全15-18张图片中角色形象、服装、妆容完全统一
4. **字幕无遮挡**：字幕绝对不遮挡女性面部、面膜主体
5. **API适配优先**：所有输出物均按COZE视频大模型API要求定义格式
