## 字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| 镜头序号 | 从1开始，顺序排列 | 1 |
| 时长（秒） | 单镜头时长，2-5秒 | 4 |
| 场景描述 | 具体场景设定 | 职场办公室 |
| 画面动作 | 具体画面动作描述（需明确人体肢体姿态） | 男性白领，短发，穿白衬衫，单侧手臂佩戴手环，自然下垂，手指舒展，手环屏幕显示心率数据 |
| 文案 | 对应分镜文案 | 心率监测，实时精准 |
| 音效备注 | 背景音乐或场景音效 | 办公室环境音 |
| 转场 | 转场效果 | 淡入、淡出淡入、科技光效、淡出 |

## 约束与注意事项

1. **角色一致性**：
   - 同一角色在所有镜头中服装、发型、外貌、风格必须完全一致
   - 禁止角色在不同镜头间突变（如衣服颜色变化、发型改变）
   - 角色描述需明确具体（年龄、性别、发型、服装、外貌特征）

2. **人体肢体规范**：
   - 画面动作描述必须明确人体肢体姿态
   - 仅展示正常成年人肢体，比例协调
   - 手臂/手部动作自然，无多余肢体、无重影、无畸形
   - 肢体与商品贴合自然（如佩戴类商品无悬浮感）

3. **时长控制**：
   - 单镜头时长：2-5秒
   - 总时长严格匹配：
     - 30秒：8-10个镜头
     - 45秒：12-15个镜头
     - 60秒：15-18个镜头

4. **转场效果**：
   - 淡入淡出+科技光效（科技风）
   - 柔焦渐变（治愈风）
   - 胶片感叠化（复古风）
   - 转场时长0.3-0.5秒
