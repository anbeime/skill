## 概览
本指南为视觉设计师提供 AI 智能配图的参考标准，确保图片与 PPT 内容高度相关，增强表达效果。

## 配图原则

### 相关性原则
- 图片必须与页面主题和内容高度相关
- 避免使用与内容无关的装饰性图片
- 图片应能够帮助观众理解内容，而非分散注意力

### 风格一致性原则
- 整个 PPT 的图片风格应保持一致
- 统一的配色方案、视觉元素、设计语言
- 避免混用过于不同的图片风格（如卡通与写实混用）

### 简洁性原则
- 图片应简洁明了，避免过于复杂的视觉元素
- 重点突出，主次分明
- 避免图片中的文字干扰内容表达

### 质量原则
- 使用高质量图片，避免模糊、失真
- 分辨率适中（推荐 1920×1080 或更高）
- 色彩鲜艳，对比度适宜

## 图片位置标注格式

### 在 JSON 数据中标注图片

在幻灯片的 `content` 数组中，使用特殊标记标注图片位置：

```json
{
  "layout": "TitleAndContent",
  "title": "市场分析",
  "content": [
    "市场规模：2023年达到100亿美元",
    "年增长率：15%",
    "[图片：市场增长趋势图]",
    "[图表：柱状图，展示近五年市场规模]"
  ],
  "notes": "建议使用柱状图展示市场规模增长趋势"
}
```

### 标注格式说明

#### 基本格式
- `[图片：图片描述]`：普通图片
- `[图表：图表类型+描述]`：图表类
- `[图标：图标名称]`：图标类

#### 图片描述要求
- 描述简洁明了，不超过 20 字
- 包含图片类型、主要元素
- 示例：
  - `[图片：商务会议场景]`
  - `[图片：团队合作讨论]`
  - `[图片：城市天际线]`

#### 图表描述要求
- 指明图表类型（柱状图、饼图、折线图等）
- 包含图表内容描述
- 示例：
  - `[图表：饼图，展示市场份额]`
  - `[图表：折线图，展示增长趋势]`
  - `[图表：流程图，展示工作流程]`

#### 图标描述要求
- 说明图标用途
- 示例：
  - `[图标：灯泡]`
  - `[图标：箭头]`
  - `[图标：对勾]`

### 多图片标注

当一页需要多张图片时：

```json
{
  "layout": "ContentWithCaption",
  "title": "产品特性",
  "content": [
    "[图片1：产品外观图]",
    "[图片2：产品界面图]",
    "[图片3：用户使用场景图]"
  ],
  "caption": "产品特性展示"
}
```

### 图片位置建议

| 布局类型 | 图片位置建议 | 标注方式 |
|---------|-------------|---------|
| TitleAndContent | 内容区任意位置 | 在 content 数组中标注 |
| ContentWithCaption | 主内容区或说明区 | 在 content 或 caption 中标注 |
| TwoColumnText | 左栏或右栏 | 使用标记区分 `[左：图片]` 或 `[右：图片]` |
| BlankSlide | 自由位置 | 在 content 数组中标注 |

## 图片使用规范

### 图片来源
- 推荐使用合法授权的图片素材库
- 注意图片版权，避免侵权
- 建议使用高质量免费图库（如 Unsplash、Pexels 等）

### 图片格式
- 推荐格式：JPG、PNG
- PNG 格式适合透明背景图片
- JPG 格式适合照片类图片

### 图片尺寸
- 标准尺寸：1920×1080（16:9）
- 最小尺寸：1280×720
- 避免使用过小的图片（< 800×600）

### 文件大小
- 单张图片建议不超过 5MB
- 整个 PPT 的图片总大小建议不超过 50MB
- 避免图片过大导致 PPT 文件过大

### 图片处理
- 保持原始比例，避免拉伸变形
- 可以适当裁剪以突出重点
- 统一风格，避免混用不同风格

## 常见问题

### Q1：如何在 PPT 中插入实际图片？
A：当前版本的脚本主要提供图片建议标注。如需插入实际图片，建议：
1. 使用 `[图片：xxx]` 标记位置
2. 生成 PPT 后手动插入图片
3. 或使用专业的 PPT 设计工具

### Q2：如何确保图片风格一致？
A：遵循以下原则：
1. 统一选择相似的图片来源和风格
2. 使用相同的配色方案
3. 保持统一的视觉元素
4. 避免混用过多不同风格的图片

### Q3：一页应该使用多少张图片？
A：建议：
- 封面页：1 张背景图
- 内容页：1-2 张图片
- 章节页：可选 1 张装饰图
- 避免一页使用过多图片导致信息过载

### Q4：图片和文字如何搭配？
A：遵循以下原则：
1. 图片应与文字内容高度相关
2. 图片位置不遮挡文字
3. 图片大小适中，不喧宾夺主
4. 文字和图片之间留有适当间距

### Q5：如何选择合适的图表类型？
A：根据数据类型选择：
- 对比数据：柱状图
- 占比数据：饼图
- 趋势数据：折线图
- 流程步骤：流程图
- 层级关系：组织结构图

### Q6：AI 智能配图的建议如何转化为实际图片？
A：建议流程：
1. 在 `[图片：xxx]` 标记中使用详细描述
2. 根据描述在免费图库中搜索
3. 选择风格一致、质量高的图片
4. 在 PPT 中替换占位图

### Q7：如何为特殊场景选择图片？
A：参考"场景化配图建议"章节，根据具体场景：
- 确定场景类型（商务/教育/学术等）
- 查看推荐图片类型
- 根据内容选择合适的图片

### Q8：图片标注是否会在生成的 PPT 中显示？
A：当前版本：
- `[图片：xxx]` 标记会在 PPT 中作为文本显示
- 用户需要手动替换为实际图片
- 未来版本可能支持直接插入图片
