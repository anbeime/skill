## 概览
本文档定义InfiniteTalk专用提示词的结构规范，确保提示词可直接作为InfiniteTalk推理脚本的`prompt`参数使用。

---

## 1. 提示词结构

### 核心结构

InfiniteTalk专用提示词结构：
```
角色固定特征 + 动作时序（帧级） + 场景环境 + 音频匹配 + 光影/氛围 + 技术约束
```

### 字段列表

| 字段 | 说明 | 是否必需 |
|-----|------|---------|
| 角色固定特征 | 描述角色的外貌、服饰、气质特征 | 必需 |
| 动作时序 | 5秒内帧级动作描述，精确到秒级 | 必需 |
| 场景环境 | 描述背景、前景、道具 | 必需 |
| 光影 | 描述色温、光照类型、光影效果 | 必需 |
| 氛围 | 描述情绪基调、整体氛围 | 必需 |
| 音频匹配 | 描述语音类型、音乐风格、音频对齐规则 | 必需 |
| 技术约束 | 描述构图比例、景深、动作强度等 | 必需 |

## 2. 字段说明

### 2.1 角色固定特征

**作用**：确保角色一致性，所有五幕使用相同描述

**描述内容**：
- 外貌特征：性别、年龄、脸型、眼睛、发型、肤色、唇色
- 服饰特征：上衣、鞋子、配饰
- 气质特征：整体气质、手部姿态、身体姿态

**格式要求**：
- 使用固定描述，严禁修改
- 仅允许调整"杏眼"的情绪表达（带笑/专注/兴奋等）

**示例**：
```
主体：25岁女性数字人，鹅蛋脸、杏眼带笑、浅棕色齐肩卷发，浅灰修身西装套裙+白衬衫+米色细跟鞋，银色简约项链；
```

### 2.2 动作时序

**作用**：描述5秒内帧级动作，确保动作连贯

**描述内容**：
- 时间分段：0-1s、1-3s、3-5s
- 动作描述：具体动作、速度、姿态
- 动作连贯：动作切换自然，无卡顿

**格式要求**：
- 精确到秒级
- 描述具体动作（抬手、转头、点触等）
- 描述动作速度（缓慢、均匀、轻柔等）
- 描述姿态变化（挺拔、端正、微倾等）

**示例**：
```
动作时序（5s）：0-1s双手轻握平板置于腰侧，1-3s缓慢抬手将平板举至胸前（速度均匀、姿态挺拔），3-5s保持举平板姿势，头部微抬看向镜头方向；
```

### 2.3 场景环境

**作用**：描述场景背景、前景、道具，确保视觉连贯

**描述内容**：
- 背景：场景类型、背景元素
- 前景：前景道具、前景元素
- 道具：关键道具描述

**格式要求**：
- 背景场景类型统一（商务场景）
- 道具描述具体（平板、文件、桌椅等）
- 背景虚化程度≥70%

**示例**：
```
场景：现代商务会议室，背景模糊大屏投影，前景简约文件；
```

### 2.4 光影

**作用**：描述色温、光照类型、光影效果，确保情绪匹配

**描述内容**：
- 色温：4000K-5500K范围
- 光照类型：侧光、顺光、顶光、逆光
- 光影效果：明暗对比、景深、虚化程度

**格式要求**：
- 色温在4000K-5500K范围内
- 光照类型明确（侧光、顺光、顶光、逆光）
- 光影效果可实现（明暗对比、景深、虚化）

**示例**：
```
光影：冷调柔光，侧光勾勒身形，明暗对比明显；
```

### 2.5 氛围

**作用**：描述情绪基调、整体氛围，确保情绪匹配

**描述内容**：
- 情绪基调：具体情绪（热情、专业、紧迫等）
- 整体氛围：整体氛围描述
- 面部表情：具体表情描述

**格式要求**：
- 情绪基调明确（热情、专业、紧迫、冷静等）
- 面部表情具体（带笑、专注、微凝、哀伤等）
- 身体动作无夸张

**示例**：
```
氛围：神秘宏大，面部表情微凝（无夸张），身体无多余晃动。
```

### 2.6 音频匹配

**作用**：描述音频类型、音乐风格、音频对齐规则，确保音频同步

**描述内容**：
- 语音类型：语速、语气
- 音乐风格：音乐类型、情绪匹配
- 音频对齐：动作节奏与音乐/语音同步

**格式要求**：
- 语音语速适中（约200-250字/分钟）
- 语气与情绪匹配（热情、专业、紧迫等）
- 音频对齐规则明确（动作匹配音乐/语音节奏）

**示例**：
```
音频类型：导购员语音（语速偏缓、语气略带悬念）+ 史诗管弦乐（低沉鼓点）；
音频对齐：抬手动作匹配鼓点重音，语音节奏与唇形同步，无肢体动作与音频断层。
```

### 2.7 技术约束

**作用**：描述构图比例、景深、动作强度等，确保生成效果

**描述内容**：
- 构图比例：9:16竖版
- 景深：虚化程度、景深范围
- 动作强度：动作强度参数（motion_strength: 0.8）
- 面部一致性：启用面部一致性管控

**格式要求**：
- 构图比例9:16竖版
- 景深≥70%（背景虚化）
- 动作强度适中（motion_strength: 0.8）
- 面部一致性启用（face_consistency: True）

**示例**：
```
技术约束：9:16竖版构图，中景固定镜头，背景虚化程度≥70%，动作强度0.8，面部一致性启用。
```

## 3. 格式要求

### 3.1 整体格式

**格式**：
```
角色固定特征 + 动作时序（帧级） + 场景环境 + 光影/氛围 + 音频匹配 + 技术约束
```

**字段分隔**：使用"；"或换行分隔各字段

**示例**：
```
主体：25岁女性数字人，鹅蛋脸、杏眼带笑、浅棕色齐肩卷发，浅灰修身西装套裙+白衬衫+米色细跟鞋，银色简约项链；
动作时序（5s）：0-1s双手轻握平板置于腰侧，1-3s缓慢抬手将平板举至胸前（速度均匀、姿态挺拔），3-5s保持举平板姿势，头部微抬看向镜头方向；
场景：现代商务会议室，背景模糊大屏投影，前景简约文件；
光影：冷调柔光，侧光勾勒身形，明暗对比明显；
氛围：神秘宏大，面部表情微凝（无夸张），身体无多余晃动。
音频类型：导购员语音（语速偏缓、语气略带悬念）+ 史诗管弦乐（低沉鼓点）；
音频对齐：抬手动作匹配鼓点重音，语音节奏与唇形同步，无肢体动作与音频断层。
技术约束：9:16竖版构图，中景固定镜头，背景虚化程度≥70%，动作强度0.8，面部一致性启用。
```

### 3.2 时间格式

**格式**：0-1s、1-3s、3-5s

**说明**：
- 精确到秒级
- 时间分段合理（0-1s、1-3s、3-5s）
- 动作描述与时间对应

### 3.3 动作描述格式

**格式**：动作 + 速度 + 姿态

**示例**：
```
缓慢抬手将平板举至胸前（速度均匀、姿态挺拔）
```

**说明**：
- 动作：抬手将平板举至胸前
- 速度：速度均匀
- 姿态：姿态挺拔

### 3.4 光影描述格式

**格式**：色调 + 光照类型 + 光影效果

**示例**：
```
冷调柔光，侧光勾勒身形，明暗对比明显
```

**说明**：
- 色调：冷调
- 光照类型：柔光、侧光
- 光影效果：勾勒身形、明暗对比明显

## 注意事项

1. **严格遵循角色固定特征**：所有五幕提示词使用相同的角色固定特征描述
2. **动作时序精确到秒级**：5秒内帧级动作描述流畅
3. **音频严格对齐**：语音节奏与唇形同步，动作节奏与音乐/语音同步
4. **光影可实现**：色温在4000K-5500K范围内，光照类型明确
5. **技术约束匹配**：构图比例9:16竖版，动作强度0.8，面部一致性启用
6. **格式统一**：字段分隔使用"；"，时间格式0-1s/1-3s/3-5s
