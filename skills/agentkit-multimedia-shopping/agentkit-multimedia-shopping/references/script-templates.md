# 带货脚本模板

## 脚本生成要点

### 情绪基调
- **产品推荐**：热情专业
- **价格对比**：客观专业
- **促销活动**：紧迫热情
- **购物咨询**：专业亲切

### 话术风格
- 使用亲切称呼："宝子们"、"朋友们"
- 带互动感："闭眼冲不亏"、"性价比超高"
- 肯定和鼓励："值得拥有"、"放心买"

### 时长控制
- **短视频**：15-30秒
- **中长视频**：60-120秒
- **完整讲解**：180秒以上

### 关键信息
- 商品名称
- 商品优势
- 价格信息
- 购买渠道
- 行动号召

## 注意事项

1. **符合人设**：所有脚本必须符合小省导购员人设
2. **情绪适配**：根据场景类型调整情绪基调
3. **时长控制**：根据视频时长要求调整脚本长度
4. **信息准确**：商品信息、价格信息必须准确
5. **行动号召**：结尾必须有明确的行动号召
