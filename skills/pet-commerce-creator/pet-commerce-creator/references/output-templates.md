## 概述

本文档定义了萌宠带货内容智创工坊中所有智能体输出物的标准化格式，确保输出物独立成档、可复制、可互动、可批量下载。

**格式设计原则**：
1. 独立标识：每个输出物有唯一标识
2. 结构清晰：内容分级、表格化、易读
3. 可复制性：支持一键复制整包内容
4. 可互动性：配置点赞/点踩/分享/评价入口
5. 批量下载：支持整包压缩下载

## 智能体输出格式

### 通用格式结构

所有智能体输出物遵循以下通用结构：

```
【输出物独立标识：智能体名称_XXX（创作主题）】
「带点赞/点踩/分享功能，互动数据实时同步至创作优化数据池」

① 核心内容1
② 核心内容2
③ 核心内容3
...

爆款+互动+带货参考标注：...

互动数据采集入口：
- [👍点赞] [👎点踩] [🔗分享]
- 评价标签：[引流] [带货] [内容]（可多选）
- 自由评价：________________________

同步至：下游智能体/品控巡检宠管家
```

### 输出物标识规范

格式：`智能体简称_XXX（创作主题）`

示例：
- `萌宠带货创意拆解包_猫粮测评带货`
- `萌宠带货故事脚本包_猫咪蹭粮袋剧情`
- `萌宠带货标准化分镜脚本_橘猫玩具测评`

### 爆款+互动+带货参考标注

格式：
```
爆款参考标注：本次创作参考了知识库中XX条爆款内容，复用了XX流量钩子、XX带货植入方法。
互动优化点：结合高互动数据，优化了XX镜头构图、XX台词句式。
带货标注：商品品类为XX，核心卖点为XX，关注引导方式为XX。
```

## 素材包格式

### 分镜图片素材包

**包结构**：
```
萌宠带货分镜图片素材包_XXX（创作主题）.zip
├── 01_猫咪扒粮袋_流量.jpg
├── 02_猫咪吃粮特写_带货.jpg
├── 03_猫咪满足舔嘴_萌点.jpg
├── 04_猫咪蹭人腿_关注.jpg
└── 图片素材说明表.txt
```

**图片素材说明表.txt**：
```
【图片素材说明表】

文件名：01_猫咪扒粮袋_流量.jpg
- 镜头序号：01
- 时长：3秒
- 场景描述：客厅地毯，阳光照射
- 画面动作：橘猫用前爪扒粮袋，尾巴轻摇
- 萌点聚焦：猫咪馋嘴，迫不及待
- 爆款参考设计：参考爆款前3秒特写镜头，橘猫形象统一
- 互动优化点：高点赞镜头，采用高饱和度暖调
- 流量带货设计：流量作用-抓眼球；带货设计-植粮品
- API适配要求：JPG格式，1080x1920，暖调柔光

文件名：02_猫咪吃粮特写_带货.jpg
- 镜头序号：02
- 时长：3秒
- 场景描述：猫碗近景

- 萌点聚焦：猫咪享受表情
- 爆款参考设计：参考爆款商品展示镜头，特写猫咪吃粮动作
- 互动优化点：高分享镜头，突出商品卖点

```
【输出物独立标识：萌宠带货创意拆解包_猫粮测评带货】
「带点赞/点踩/分享功能，互动数据实时同步至创作优化数据池」

① 核心创作定位：
- 创作类型：好物测评
- 带货品类：主粮（猫粮）
- 涨粉目标：精准宠主
- 核心流量钩子：前3秒猫咪从粮袋钻出（爆款参考）
- 商品核心卖点：配方干净、无添加、适口性好
- 关注引导方式：关注领优惠券

② 核心萌点提炼：
- 萌点1：猫咪馋嘴扒粮袋，尾巴轻摇（结合爆款萌点+高互动萌点）
  - 萌点与流量钩子结合：前3秒展示，抓眼球
  - 萌点与商品卖点结合：扒粮袋动作暗示猫粮好吃
  - 对应情节节点：镜头01（0-3秒）
- 萌点2：猫咪吃粮特写，满足表情（爆款参考）
  - 萌点与流量钩子结合：特写镜头提完播
  - 萌点与商品卖点结合：吃粮动作展示适口性
  - 对应情节节点：镜头02（3-6秒）
- 萌点3：猫咪满足舔嘴，蹭人腿（高互动萌点）
  - 萌点与流量钩子结合：蹭人互动引导关注
  - 萌点与商品卖点结合：满足表情暗示猫粮好吃

  - 穿搭：白色T恤+牛仔裤，休闲风格
