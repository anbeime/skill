## 技能调用链速查

### 场景A：网页采集类

#### A1：采集并发布
```
url-to-markdown → format → illustrator → publish
```
**时间**：5-8分钟
**输出**：Markdown文件 + 配图 + 草稿箱链接

#### A2：采集并配图
```
url-to-markdown → illustrator
```
**时间**：3-5分钟
**输出**：Markdown文件 + 配图

#### A3：采集转小红书
```
url-to-markdown → xhs-images
```
**时间**：4-6分钟
**输出**：小红书图文系列（1-10张）

### 场景B：热点创作类

#### B1：热点文章生成
```
wechat-hotspot-publisher
```
**时间**：3-5分钟
**输出**：HTML文章 + 草稿箱链接

#### B2：热点图文生成
```
wechat-hotspot → xhs-images
```
**时间**：5-7分钟
**输出**：HTML文章 + 小红书图文

#### B3：热点多平台发布
```
wechat-hotspot → xhs-images → multi-publish
```
**时间**：6-10分钟
**输出**：多平台发布状态

## 常见问题速查

### Q1: 如何指定场景？
**A**: 在描述中明确说明，如"执行场景A1"或"采集并发布"

### Q2: 如何同时发布到多个平台？
**A**: 在描述中列出平台，如"发布到微信和小红书"

### Q3: 如何自定义小红书风格？
**A**: 在描述中指定，如"使用notion风格"或"知识卡片风格"

### Q4: 网页需要登录怎么办？
**A**: 说明"需要登录"，系统会使用等待模式

### Q5: 如何批量处理？
**A**: 提供多个URL或文件，如"采集这5篇文章"

### Q6: 执行失败怎么办？
**A**: 查看错误信息，检查参数，或单独测试各技能

### Q7: 如何取消执行？
**A**: 随时可以中断，已完成的步骤会保留

### Q8: 如何查看生成的文件？
**A**: 系统会提供文件路径，可直接打开

### Q9: 发布到微信后在哪里查看？
**A**: 登录微信公众号后台的"草稿箱"

### Q10: 小红书图文如何发布？
**A**: 图文生成后需手动上传到小红书

## 使用技巧

### 技巧1：一次性提供所有参数
```
✓ "采集 https://example.com/article 并发布到微信公众号"
✗ "采集文章" → 需要多次询问
```

### 技巧2：明确指定平台
```
✓ "发布到微信和小红书"
✗ "发布" → 需要询问平台
```

### 技巧3：使用风格关键词
```
✓ "生成知识卡片风格的小红书图文"
✗ "生成小红书图文" → 使用默认风格
```

### 技巧4：说明特殊需求
```
✓ "采集这个需要登录的网页"
✗ "采集网页" → 可能采集失败
```

### 技巧5：分步执行复杂任务
```
✓ "先采集并配图，满意后再发布"
✗ "采集并发布" → 无法中途调整
```

## 注意事项

### ⚠️ 发布前检查
- 内容是否符合平台规范
- 图片是否清晰可读
- 标题是否吸引人
- 标签是否准确

### ⚠️ 版权注意
- 采集的内容注意版权
- 图片使用注意授权
- 标注原文来源

### ⚠️ 平台规则
- 微信：草稿箱模式，需人工确认
- 小红书：图文需手动上传
- X：需要浏览器登录状态
- B站：需要账号权限

### ⚠️ 技术限制
- 部分网站有反爬虫
- 网络环境影响速度
- Chrome依赖部分功能
