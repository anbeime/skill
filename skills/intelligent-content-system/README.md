# 智能内容创作与发布系统

> 一句话完成从内容获取到多平台发布的全流程

## 简介

智能内容创作与发布系统是一个整合了5个核心技能的全流程自动化系统，通过智能场景识别和工作流编排，让用户用自然语言描述需求即可完成复杂的内容创作和发布任务。

## 核心特性

- **🎯 智能场景识别**：自动识别14种常见场景，无需了解技术细节
- **🔄 自动工作流编排**：智能调用5个子技能，实现端到端自动化
- **📱 多平台支持**：支持微信公众号、小红书、X/Twitter、B站
- **⚡ 实时进度反馈**：每步执行都有清晰的进度提示
- **🎨 灵活风格定制**：支持10种小红书视觉风格和8种布局

## 支持场景

### 网页内容采集与再创作（A类）
- **A1**：采集并发布 - 采集网页→优化→配图→发布
- **A2**：采集并配图 - 采集网页→智能配图
- **A3**：采集转小红书 - 采集网页→生成小红书图文

### 热点内容创作（B类）
- **B1**：热点文章生成 - 采集热点→AI生成文章→发布
- **B2**：热点图文生成 - 采集热点→生成小红书图文
- **B3**：热点多平台发布 - 采集热点→多平台分发

### 原创内容优化与发布（C类）
- **C1**：文章配图 - 分析文章→生成配图
- **C2**：转小红书图文 - 文章→小红书图文系列
- **C3**：多平台发布 - 优化→多平台分发

### 单一功能调用（D类）
- **D1**：仅采集网页
- **D2**：仅配图
- **D3**：仅小红书图文
- **D4**：仅发布

## 整合的技能

本系统整合了以下5个核心技能：

1. **baoyu-url-to-markdown** - 网页内容采集
   - 使用Chrome CDP技术
   - 支持JS渲染和登录页面
   - 输出干净的Markdown格式

2. **article-illustrator** - 文章智能配图
   - 分析文章结构
   - 识别配图位置
   - 生成风格统一的插图

3. **baoyu-xhs-images** - 小红书图文生成
   - 10种视觉风格
   - 8种信息布局
   - 1-10张图文系列

4. **wechat-hotspot-publisher** - 热点内容创作
   - 智能热点采集
   - AI爆款文案生成
   - 多平台发布支持

5. **content-creation-publisher** - 内容创作全流程
   - 整合上述技能
   - 提供完整工作流
   - 多平台发布管理

## 使用方式

### 方式1：自然语言描述（推荐）
直接用自然语言描述需求，系统会自动识别场景并执行：

```
"采集这篇文章并发布到微信公众号 https://example.com/article"
"根据AI热点生成小红书图文"
"给这篇文章配图 article.md"
"把这个网页转成小红书图文 https://example.com/tutorial"
```

### 方式2：指定场景代码
如果熟悉场景代码，可以直接指定：

```
"执行场景A1，URL是 https://example.com/article，发布到微信"
"执行场景B2，关键词是AI工具"
"执行场景C1，文件是 article.md"
```

### 方式3：分步执行
对于复杂需求，可以分步执行：

```
"先采集这篇文章并配图"
→ 查看结果，满意后继续
"把刚才的文章发布到微信"
```
