## 概览
本文档说明如何将识别后的文字改写为不同格式，包括邮件、笔记、社交媒体文案等，提供改写规则和模板。

## 改写类型说明

### 邮件改写
**适用场景**：语音备忘、会议纪要、工作沟通

**改写原则**：
- 正式、专业
- 结构清晰（主题、正文、落款）
- 语气礼貌
- 信息完整

**核心要素**：
- 主题：简明扼要，概括邮件内容
- 称呼：根据收件人选择合适的称呼
- 正文：逻辑清晰，分段表达
- 结尾：礼貌用语
- 落款：姓名、职位、日期

### 笔记改写
**适用场景**：会议纪要、学习笔记、灵感记录

**改写原则**：
- 要点清晰
- 层次分明
- 使用列表和标记
- 便于快速浏览

**核心要素**：
- 标题：明确笔记主题
- 分类：按主题或时间分类
- 要点：使用项目符号或数字编号
- 标记：使用标记突出重点
- 时间：记录时间信息

### 社交媒体改写
**适用场景**：朋友圈、微博、小红书等

**改写原则**：
- 简洁明了
- 有吸引力
- 使用表情符号
- 添加话题标签

**核心要素**：
- 开头：吸引注意力的开场
- 正文：简洁有趣的内容
- 表情符号：增强表达
- 话题标签：增加曝光
- 结尾：号召行动或互动

## 邮件改写规则

### 基础模板
```
主题：[邮件主题]

[收件人称呼]：

[开场白]

[正文内容，分段表达]

[结尾]

[落款]
```

### 语气调整
- **正式邮件**：使用"您好"、"请"、"谢谢"等礼貌用语
- **内部邮件**：可以更直接，但保持基本礼貌
- **请求邮件**：语气要客气，说明请求原因

### 常用句式
- 开场白：
  - "您好！"
  - "关于..."
  - "首先感谢..."
  
- 正文过渡：
  - "另外，"
  - "此外，"
  - "需要注意的是，"
  
- 结尾：
  - "如有疑问，请随时联系我。"
  - "期待您的回复。"
  - "祝好！"

### 示例

**原始语音**：
"小王，关于明天的项目会议，请准备好以下材料：项目进度报告、预算表、团队人员名单。会议时间是上午10点，地点在3号会议室。"

**改写为邮件**：
```
主题：明天项目会议准备材料

小王：

你好！

关于明天上午10点在3号会议室召开的项目会议，请提前准备好以下材料：

1. 项目进度报告
2. 预算表
3. 团队人员名单

如需更多信息，请随时联系我。

祝好！

张经理
```

## 笔记改写规则

### 基础模板
```
[笔记标题]

## 时间地点
- 时间：[时间]
- 地点：[地点]

## 参会人员
- [人员1]
- [人员2]

## 主要内容
1. [要点1]
   - [细节]
   
2. [要点2]
   - [细节]

## 待办事项
- [ ] [任务1]
- [ ] [任务2]
```

### 结构组织
- **按时间**：按时间顺序记录要点
- **按主题**：按主题分类整理要点
- **按重要性**：优先级排序，重要内容置顶

### 标记使用
- **重点**：使用 ⭐ 标记重要内容
- **待办**：使用 📝 标记待办事项
- **提醒**：使用 ⚠️ 标记需要注意的内容

### 示例

**原始语音**：
"今天开会讨论了Q1的业绩，销售额增长了20%，新用户增加了5000人。Q2要推出新产品线，加强市场推广，优化客户服务。批准了新产品线的预算，成立了市场推广小组。"

**改写为笔记**：
```
# Q1 业绩总结与 Q2 工作计划

## 时间
- 2024年1月23日

## 业绩总结
- 📊 销售额增长：+20%
- 👥 新用户增加：+5000人

## Q2 工作计划
1. ⭐ 推出新产品线
   - 预算已批准
   
2. 加强市场推广
   - 成立市场推广小组
   
3. 优化客户服务
   - 提升服务质量

## 决议事项
- ✅ 批准新产品线预算
- ✅ 成立市场推广小组

## 待办事项
- [ ] 制定新产品线推广计划
- [ ] 招募市场推广人员
```

## 社交媒体改写规则

### 基础模板
```
[吸引注意力的开头]

[正文内容，简洁有趣]

[表情符号]

[话题标签]

[结尾：号召行动或互动]
```

### 表情符号使用
- 积极：🌟 ✨ 💪 👍
- 美食：☕ 🍰 🍕 🍔
- 旅行：🌍 ✈️ 🏖️ 📸
- 工作：💼 💻 📝 📊

### 话题标签
- 通用：#日常生活 #分享 #推荐
- 美食：#美食探店 #咖啡 #周末好去处
- 工作：#工作日常 #效率 #学习
- 旅行：#旅行 #拍照 #打卡

### 示例

**原始语音**：
"今天尝试了一家超棒的咖啡店，咖啡味道很浓郁，环境也很舒服，很适合工作。推荐给大家！"

**改写为社交媒体文案**：
```
☕️ 今日份咖啡推荐！

今天发现了一家宝藏咖啡店 ☕✨

咖啡口感浓郁，环境舒适超治愈，简直是工作充电的好地方～

#咖啡探店 #工作日常 #周末好去处

📍 地址：[咖啡店名称]
⭐ 推荐：招牌拿铁、手冲咖啡

大家还有推荐的好去处吗？评论区分享一下吧！👇
```

## 改写最佳实践

### 1. 保持原意
- ✅ 保留核心信息和要点
- ✅ 不随意增删内容
- ❌ 不要改变原始意图

### 2. 调整语气
- 根据目标场景选择合适的语气
- 邮件：正式、礼貌
- 笔记：客观、清晰
- 社交媒体：轻松、活泼

### 3. 优化结构
- 邮件：主题、正文、落款
- 笔记：标题、分类、要点
- 社交媒体：开头、正文、话题标签

### 4. 增强可读性
- 使用列表、分段
- 添加适当的标记
- 使用表情符号（社交媒体）

### 5. 适配平台
- 了解目标平台的特点
- 遵循平台的格式规范
- 考虑平台的阅读习惯

### 6. 持续优化
- 根据反馈调整改写风格
- 积累常用的模板和句式
- 关注新的流行语和话题标签
