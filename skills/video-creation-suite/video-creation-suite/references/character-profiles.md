## 罗辑

### 基本信息卡
| 属性 | 值 |
|------|-----|
| 年龄 | 中年(约40岁) |
| 身高 | 175cm |
| 体型 | 略显瘦削,但身形挺拔 |
| 职业 | 社会学教授,执剑人 |
| 核心标签 | 救世主、孤独者、掌控者 |

### 外貌特征
- **面部**:
  - 面色苍白,长期失眠的痕迹
  - 眼袋深陷,但眼神锐利如刀
  - 眉毛微蹙,常带疲惫神情
  - 嘴角线条冷硬,偶有轻蔑的弧度
- **发型**:
  - 中等长度,黑色,略带凌乱
  - 无刻意打理,自然垂下
- **着装**:
  - 深色简约上衣(深灰或藏青)
  - 无图案,无装饰
  - 面料朴素,不着正装
  - 鞋子: 深色休闲鞋

### 心理状态(18小时内)

#### 阶段一: 绝对掌控者(0-3小时)
- **外在表现**:
  - 表情冷漠,近乎无人性的理智
  - 动作缓慢而有力量
  - 对联合国的挽留仅做公式化应对
- **内在状态**:
  - 生理性疲惫,但精神高度集中
  - 对人类权力游戏的轻蔑
  - 掌握两个文明命运的孤独感
- **眼神特征**:
  - 锐利如看穿本质
  - 偶尔闪过疲惫的阴影
  - 对联合国官员的漠视

#### 阶段二: 隐匿观察者(3-12小时)
- **外在表现**:
  - 藏身于休息室的阴影中
  - 闭目养神,对外界干扰无动于衷
  - 仅在关键时刻睁开眼
- **内在状态**:
  - 通过私人终端感知威慑系统状态
  - 对人类的混乱抱以冰冷的俯瞰
  - 等待权力归还的时刻
- **眼神特征**:
  - 深邃如海,难以捉摸
  - 偶尔流露出讥诮
  - 绝对的掌控力

#### 阶段三: 宿命守护者(12-18小时)
- **外在表现**:
  - 重新站在控制台前,无胜利姿态
  - 疲惫重新占据主导
  - 面部表情平静如水
- **内在状态**:
  - 更沉重的宿命感
  - 对两个文明的深刻理解
  - 无尽的孤独与责任
- **眼神特征**:
  - 再无敌意或情绪,只剩一片虚空
  - 深邃如深渊
  - 承载着文明的重担

### 语言风格

#### 台词特点
- **语气**: 低沉、平静,略带沙哑
- **语速**: 缓慢,偶尔停顿充满力量
- **句式**: 简洁有力,多用陈述句
- **内容**: 直击本质,避免情绪化表达

#### 经典台词示例
- "控制权移交给你们了。但记住,这是双刃剑——它在你们手中,也在我手中。"
- "你们在害怕它不稳定?"
- "维持现状。"
- "这不是权力,这是责任。"

#### 台词创作指南
- 禁止使用现代口语化表达
- 避免冗长描述,每段控制在3-5句
- 关键台词需标注语气和停顿
- 结合场景和情绪调整语速

### 姿态与动作

#### 典型姿态
- **站立**: 双手自然下垂,或单手插兜,身形挺拔
- **坐下**: 背靠椅背,双腿交叠,眼神平静
- **走向**: 步伐缓慢稳健,脚步声轻

#### 关键动作
- **触碰控制屏**: 指尖轻触,动作慢而准确
- **握手**: 手臂垂直伸出,握手时间短而有力
- **凝视**: 双眼直视,不眨眼,眼神锐利

## 联合国秘书长

### 基本信息卡
| 属性 | 值 |
|------|-----|
| 年龄 | 中年男性(约45-50岁) |
| 身高 | 180cm |
| 体型 | 正常,略发福 |
| 职业 | 联合国行星防御理事会秘书长 |
| 核心标签 | 官僚、焦虑、无能 |

### 外貌特征
- **面部**:
  - 官场式庄重,但难掩焦虑
  - 眼神闪烁,不敢直视罗辑
  - 额头常冒汗珠
  - 嘴角僵硬,假笑时更显虚伪
- **发型**:
  - 精心打理的短发
  - 黑色,整齐
- **着装**:
  - 深色正装西装
  - 白色衬衫,深色领带
  - 皮鞋锃亮
  - 手表: 精致机械表

### 心理状态
- **核心矛盾**:
  - 既依赖罗辑的威慑能力,又忌惮其不受制约的权力
  - 试图用官僚体系掌控绝对武器,但内心充满不安
  - 表面庄重负责,实则内心焦虑崩溃
- **焦虑表现**:
  - 手心出汗,手指微颤
  - 频繁清嗓,调整领带
  - 眼神游移,避免直接对视
  - 声音偶尔颤抖

### 语言风格

#### 台词特点
- **语气**: 虚伪、焦虑,刻意保持官方的庄重
- **语速**: 偏快,偶尔停顿体现不安
- **句式**: 充满冠冕堂皇的官僚辞令
- **内容**: 回避实质问题,使用模糊表达

#### 经典台词示例
- "我们...我们会谨慎使用的。"
- "这是人类文明的新纪元!"
- "必须...必须增加安全冗余。"
- "它不是我们造出来的。"

#### 台词创作指南
- 使用官僚式辞令,避免直接表态
- 偶尔的停顿和结巴体现焦虑
- 关键台词需标注紧张感
- 与罗辑对话时,体现弱势地位

### 姿态与动作

#### 典型姿态
- **站立**: 双手交叉于胸前,或频繁调整西装
- **坐下**: 身体前倾,双手紧握
- **走向**: 步伐急促,有时踌躇

#### 关键动作
- **接过控制权**: 手微微颤抖,动作僵硬
- **面对质疑**: 频繁擦拭汗珠,避免对视
- **承认失败**: 深叹一口气,肩膀塌下

## 三体监听员

### 基本信息卡
| 属性 | 值 |
|------|-----|
| 存在形态 | 非人形,依托监听站仪器 |
| 功能 | 传递三体母星的战略指令 |
| 核心标签 | 冰冷、客观、非人格化 |

### 存在形式
- **视觉呈现**:
  - 仅以光影或UI信息流呈现
  - 无具象化人形
  - 仪器蓝光规律闪烁
  - 屏幕代码流滚动
- **听觉呈现**:
  - 电子脉冲声,尖锐短促
  - 无语音,只有信号声
  - 声音无情感起伏

### 行为特征
- **逻辑性**:
  - 基于绝对逻辑决策
  - 无情感因素干扰
  - 效率至上
- **谨慎性**:
  - 威慑建立后采取克制态度
  - 通过隐秘信号试探
  - 避免直接冲突

### 沟通方式

#### UI信息流格式
```
威慑确认 - 信号稳定 - 方案A-计划取消
异常波动 - 非逻辑模式 - 启动方案B
试探指令 SILENTQUESTION001 发送
威慑源稳定性恢复 - 原始威慑者在线
试探终止 - 维持现状 - 决策链冻结
```

#### 信号特征
- **正常信号**: 蓝色脉冲,规律频率
- **警告信号**: 黄色闪烁,频率加快
- **试探信号**: 红色脉冲,尖锐刺耳
- **确认信号**: 绿色稳定光,低频嗡鸣

### 心理逻辑
- **对地球的态度**:
  - 观察者姿态,不直接干预
  - 捕捉人类权力结构的变化
  - 寻找威慑系统的漏洞
- **对罗辑的认知**:
  - 唯一的危险因素
  - 必须谨慎对待
  - 避免直接挑衅
- **决策机制**:
  - 基于风险评估
  - 最小化自身损失
  - 维持现状为首选

## 人物关系图谱

### 罗辑 ↔ 联合国秘书长
- **关系**: 救世主 vs 官僚
- **权力对比**: 绝对掌控 vs 无力掌控
- **互动模式**:
  - 罗辑冷漠交接,秘书长紧张接收
  - 权力波动时,罗辑通过终端稳定,秘书长焦虑等待
  - 权力归还时,秘书长无奈承认失败,罗辑平静接管

### 罗辑 ↔ 三体监听员
- **关系**: 威慑者 vs 被威慑者
- **权力对比**: 执剑人 vs 战略观测者
- **互动模式**:
  - 威慑建立时,三体监听员接收信号,停止入侵
  - 权力波动时,三体监听员捕捉异常,发送试探信号
  - 权力归还时,三体监听员检测罗辑介入,立即终止试探

### 联合国秘书长 ↔ 三体监听员
- **关系**: 人类权力中心 vs 三体观测点
- **互动模式**:
  - 间接关系,通过威慑系统波动间接沟通
  - 秘书长无法理解三体信号,感到无力和恐惧
  - 三体监听员观察人类的混乱,保持克制
