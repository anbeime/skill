# 分镜脚本标准模板

## 基本信息区
| 字段 | 说明 |
|------|------|
| 场次 | 编号(如: S01、S02) |
| 场景名称 | 地点(如: 威慑控制室、联合国会议厅、三体监听站) |
| 时长 | 该场景总时长(秒) |
| 情绪基调 | 情绪描述(如: 压抑、紧张、焦虑) |

## 分镜格式

### 单个镜头格式

```
镜头编号 | 镜头类型 | 时长 | 运镜 | 光影 | 核心动作 | 音效/台词 | 备注
```

#### 字段说明

| 字段 | 可选值 | 说明 |
|------|--------|------|
| 镜头编号 | L01、L02... | 顺序编号 |
| 镜头类型 | 特写/中景/全景/移动镜头 | 镜头范围 |
| 时长 | 0.5-5秒 | 镜头持续时间 |
| 运镜 | 固定/推拉/切镜/摇摄 | 镜头运动方式 |
| 光影 | 高反差/柔光/暗调/点光 | 光线效果 |
| 核心动作 | 人物/画面动作描述 | 视觉内容 |
| 音效/台词 | 音效描述或台词文本 | 听觉内容 |
| 备注 | 转场/特效说明 | 补充信息 |

## 分镜创作指南

### 三幕式结构

#### 第一幕: 铺垫(0-3分钟)
- 节奏: 慢节奏,长镜头
- 目的: 建立仪式感,展示权力移交的庄严
- 镜头: 多用中景和全景,少用特写
- 转场: 淡入淡出为主

#### 第二幕: 冲突(3-6分钟)
- 节奏: 快节奏,短镜头
- 目的: 展现权力博弈和文明试探
- 镜头: 多用特写和切镜,蒙太奇手法
- 转场: 硬切为主,增强紧张感

#### 第三幕: 高潮(6-8分钟)
- 节奏: 递进节奏,长短镜头结合
- 目的: 展现权力归还和文明对峙
- 镜头: 特写+全景切换,形成对比
- 转场: 淡出为主,营造宿命感

### 镜头节奏控制

| 阶段 | 平均镜头时长 | 特写比例 | 快切使用 | 整体节奏 |
|------|-------------|---------|---------|---------|
| 第一幕 | 2-3秒 | 20% | 极少 | 慢 |
| 第二幕 | 0.5-1秒 | 50% | 频繁 | 快 |
| 第三幕 | 1-2秒 | 30% | 适中 | 中等 |

### 关键场景分镜要点

#### 权力移交场景
- **焦点**: 罗辑的手指、眼神,联合国官员的紧张表情
- **镜头**: 特写+中景交替,强调交接的仪式感
- **光影**: 高反差,突出罗辑的孤独掌控
- **音效**: 威慑系统声作为背景音,低沉压抑

#### 系统波动场景
- **焦点**: 屏幕上的波动、仪器的闪烁、人员的慌乱
- **镜头**: 快速切镜,蒙太奇手法
- **光影**: 暗调+黄色警告光,制造紧张感
- **音效**: 尖锐脉冲声,不和谐音效

#### 权力归还场景
- **焦点**: 秘书长的无奈,罗辑的平静,系统的稳定
- **镜头**: 长镜头+特写,形成对比
- **光影**: 柔和光线,象征秩序恢复
- **音效**: 恢复规律脉冲,氛围安静

## 输出要求

### 格式规范
- 文件格式: Markdown (.md)
- 编码: UTF-8
- 命名: `分镜脚本-[场次名称].md`

### 内容要求
- 每个场景必须包含基本信息区
- 镜头数量: 每个场景5-10个镜头
- 总时长: 符合8-10分钟视频节奏
- 备注: 标注转场、特效等关键技术点

### 质量检查
- [ ] 镜头类型、时长、运镜是否符合视觉风格指南
- [ ] 光影效果是否与场景色调一致
- [ ] 音效/台词是否符合人物性格
- [ ] 转场方式是否符合严肃基调
- [ ] 整体节奏是否符合三幕式结构
