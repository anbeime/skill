# 提示词模板

## 视觉分析提示词

### 完整视觉分析
```
详细分析这张图片的以下维度:
1. 场景类型(室内/室外/特定场景)
2. 人物特征(数量、外貌、表情、动作)
3. 构图方式(特写/中景/全景/镜头角度)
4. 光影效果(光源方向、光线强度、阴影)
5. 色调体系(主色调、配色方案)
6. 材质质感(光滑/粗糙/细腻)
7. 整体风格(写实/抽象/科技/复古/艺术流派)

输出JSON格式,包含以上所有维度。
```

### 风格特征提取
```
提取这张图片的核心风格特征:
1. 视觉风格关键词(5-10个)
2. 主色调及色彩搭配
3. 构图特点和镜头语言
4. 光影氛围
5. 整体调性(冷峻/温暖/压抑/欢快等)

输出JSON格式,便于后续风格迁移。
```

## 图片生成提示词

### 保持风格但改变内容
```
生成一张[新场景描述]的图片,要求:
1. 保持以下风格特征: [从分析结果提取的色调/构图/光影/风格]
2. 画面内容为: [新场景的具体描述]
3. 构图方式: [与原视频相似]
4. 光影效果: [与原视频相似]
5. 整体质感: [与原视频相似]

分辨率: 1920x1080或1080x1920
风格: [原视频风格]
```

### 风格迁移提示词模板
```
原视频风格: [风格描述,如"冷灰色调、低角度拍摄、硬质光、科技感"]

新场景: [新场景描述]

生成提示词:
"[新场景的详细描述],保持{冷灰色调}的风格,使用{低角度拍摄}的构图,光线{硬质光},整体呈现{科技感}的氛围。"
```

## 旁白生成提示词

### 基于新场景的旁白
```
根据以下场景描述生成旁白文本:

新场景: [场景描述]
风格要求: [与原视频一致的旁白风格,如"冷静客观/温暖亲切/悬疑紧张"]
时长要求: [对应镜头的时长]

旁白要求:
1. 内容与新场景匹配
2. 语言风格保持一致
3. 语句简洁有力,适合配音
4. 每段旁白控制在[X]字以内

输出格式:
[场景序号] [旁白文本]
```

### 批量旁白生成
```
根据以下新场景列表,批量生成旁白文本:

场景列表:
[场景1]: [描述]
[场景2]: [描述]
...

旁白风格: [整体风格]
每段时长: [秒数要求]

生成旁白文本,格式为:
[1] [旁白文本]
[2] [旁白文本]
...
```

## 音效设计提示词

### 场景音效提示词
```
为以下场景设计音效方案:

场景: [场景描述]
环境: [室内/室外/特定环境]
氛围: [紧张/轻松/神秘/科技等]

音效要求:
1. 背景环境音(持续的低频噪音/环境声)
2. 关键动作音效(按键声/脚步声等)
3. 音量比例: 环境音XX%,动作音效XX%
4. 音效风格: [科技感/自然感/电子感等]

输出音效方案JSON。
```

### 背景音乐提示词
```
为以下视频选择背景音乐:

视频风格: [原视频风格]
情绪基调: [紧张/轻松/神秘/科技等]
节奏要求: [快/中/慢]

背景音乐要求:
1. 风格: [电子/钢琴/弦乐/混合]
2. 节奏: [与视频节奏匹配]
3. 音量: 不盖过旁白和关键音效
4. 时长: 与视频时长一致

输出音乐风格描述和推荐。
```

## 字幕生成提示词

### 基于旁白的字幕
```
根据以下旁白文本生成字幕:

旁白文本: [旁白内容]
视频时长: [总时长]
镜头数量: [镜头数]

字幕要求:
1. 文本与旁白一致,无错别字
2. 每段字幕长度适中(每秒1-2字)
3. 位置: 底部居中,避免遮挡关键元素
4. 字体: 简洁易读的无衬线字体
5. 颜色: 与背景对比度高

输出SRT格式字幕文件。
```

### 字幕时间轴配置
```
根据以下旁白和镜头时长,配置字幕时间轴:

旁白文本:
[1] "第一段旁白"
[2] "第二段旁白"
...

镜头时长:
[1] 3.0秒
[2] 4.0秒
...

要求:
1. 字幕开始时间与镜头同步
2. 字幕持续时间=镜头时长-0.5秒
3. 转场时段不显示字幕

输出SRT格式字幕,包含精确的时间码。
```

## 转场设计提示词

### 转场效果选择
```
为以下相邻场景设计转场效果:

场景A: [描述]
场景B: [描述]
场景关系: [相似/对比/递进/转场关系]

转场要求:
1. 转场类型: 淡入淡出/切镜/溶镜/推拉镜头
2. 转场时长: 0.3-0.5秒
3. 转场效果: 自然流畅,无卡顿

输出转场方案JSON。
```

### 批量转场配置
```
根据以下场景列表,批量设计转场效果:

场景列表:
[1] [场景描述]
[2] [场景描述]
[3] [场景描述]
...

转场原则:
1. 相似场景用淡入淡出
2. 对比场景用切镜
3. 递进场景用溶镜
4. 每个转场0.3-0.5秒

输出转场配置JSON。
```

## 完整二创流程提示词

### 阶段1: 风格分析与提取
```
分析原视频抽帧图片,提取以下信息:
1. 视觉风格关键词
2. 主色调和色彩体系
3. 构图特点和镜头语言
4. 光影效果和氛围
5. 整体调性和风格流派

输出风格分析报告JSON,用于后续素材生成。
```

### 阶段2: 新场景设计与提示词生成
```
基于风格分析结果,设计新场景:

原风格: [从阶段1提取]
新场景主题: [用户指定的主题或智能体自创]

设计要求:
1. 保持原风格特征
2. 改变具体场景内容
3. 保持画面节奏和结构

输出新场景设计JSON,包含每个场景的详细描述和图片生成提示词。
```

### 阶段3: 旁白与字幕创作
```
基于新场景设计,创作旁白和字幕:

新场景: [从阶段2获取]
旁白风格: [与原视频一致]

创作要求:
1. 旁白内容与新场景匹配
2. 语言风格保持一致
3. 字幕与旁白精确同步

输出旁白文本和字幕文件。
```

### 阶段4: 音效与音乐设计
```
基于新场景和旁白,设计音效和音乐:

新场景: [从阶段2获取]
旁白: [从阶段3获取]
原风格: [从阶段1获取]

设计要求:
1. 场景音效与新场景匹配
2. 背景音乐保持原风格
3. 音量比例合理

输出音效方案和音乐选择。
```

### 阶段5: 视频合成配置
```
整合所有素材,生成视频合成配置:

素材:
- 新图片: [路径]
- 旁白音频: [路径]
- 音效: [路径]
- 背景音乐: [路径]
- 字幕: [路径]

配置要求:
1. 设计转场效果
2. 匹配音频与视频时长
3. 确保整体风格统一

输出合成配置JSON。
```
