## 威慑控制系统界面

### 整体风格
- 风格: 极简主义,功能主义
- 主色调: 冷灰(#333333) + 科技蓝(#0099FF)
- 字体: 思源黑体(中文), Roboto(英文)
- 布局: 居中对称,模块化

---

### 核心界面元素

#### 1. 威慑触发按钮

**位置**: 界面中央,最醒目位置
**尺寸**: 120x120px,圆形
**颜色**: 红色(#FF0000),边缘发光
**状态**:
- 待机: 红色,暗淡
- 激活: 红色,高亮,脉冲闪烁
- 触发: 红色,持续高亮

**视觉描述**:
```
圆形红色按钮,位于界面正中央
边缘有红色光晕,向外扩散
按钮中心有威慑标志(宇宙坐标符号)
上方文字: "威慑触发" (红色,粗体,字重700)
下方状态: "待机" / "激活" / "已触发"
```

---

```
  星系编号: Alpha Centauri

```

---

#### 3. 信号强度指示器

**位置**: 顶部,横跨整个界面
**尺寸**: 1920x60px
**颜色**: 蓝色渐变背景

**视觉描述**:

## 三体监听站界面

### 整体风格
- 风格: 极简主义,数据驱动
- 主色调: 纯黑(#000000) + 深蓝(#0066CC)
- 字体: Fira Code (等宽字体)
- 布局: 垂直滚动,信息流

---

### 核心界面元素

#### 1. 信息流显示区

**位置**: 界面中央,垂直滚动
**尺寸**: 1200x800px
**颜色**: 黑色背景,白色/青绿色文字

**视觉描述**:
```
滚动方式: 从下至上,匀速滚动
字体: 等宽,绿色(#00FF00)
格式: 时间戳 + 指令内容

信息流示例:
[00:00:00:001] 威慑确认
[00:00:05:002] 信号稳定
[00:00:10:003] 方案A-计划取消
[02:35:42:156] 异常波动
[02:35:47:001] 非逻辑模式
[02:35:52:003] 启动方案B
[02:36:00:001] 试探指令发送
[05:12:33:876] 威慑源稳定性恢复
[05:12:35:002] 原始威慑者在线
[05:12:40:005] 试探终止
[05:12:45:008] 维持现状
[05:12:50:000] 决策链冻结
```

---

#### 2. 频率分析波形图

**位置**: 界面右侧,垂直排列
**尺寸**: 300x400px
**颜色**: 蓝色波形,黑色背景

**视觉描述**:
```
类型: 实时频谱分析
横轴: 频率 (Hz)
纵轴: 幅度
颜色: 蓝色波形 (#0066CC)

正常状态: 稳定波形,低频为主
波动状态: 波形震荡,频率加快
试探信号: 尖锐波峰,高频出现
```

---

#### 3. 状态指示灯

**位置**: 界面顶部,横向排列
**尺寸**: 每个50x50px,圆形
**颜色**: 红/黄/绿

**视觉描述**:
```
左侧: "系统状态" (白色文字)

指示灯1 (威慑确认):
  颜色: 绿色
  状态: 亮起 (威慑建立)

指示灯2 (信号监测):
  颜色: 蓝色
  状态: 闪烁 (信号波动)

指示灯3 (试探指令):
  颜色: 红色
  状态: 亮起 (发送试探)

指示灯4 (维持现状):
  颜色: 绿色
  状态: 亮起 (决策冻结)
```

---

#### 4. 脉冲灯阵列

**位置**: 界面底部,横向排列
**尺寸**: 每个30x30px,圆形,共20个
**颜色**: 蓝色脉冲光

**视觉描述**:
```
脉冲模式:
正常: 左到右顺序闪烁,稳定频率
波动: 随机闪烁,频率加快
试探: 快速闪烁,频率极快
恢复: 顺序闪烁,频率恢复正常

颜色变化:
正常: 蓝色 (#0099FF)
波动: 黄色 (#FFFF00)
试探: 红色 (#FF0000)
恢复: 蓝色 (#0099FF)
```

## 通用UI元素

### 字体规范

#### 中文字体
- **主字体**: 思源黑体 / 冬青黑体
- **标题**: 粗体,字重700
- **正文**: 常规,字重400
- **代码**: 等宽,字重400

#### 英文字体
- **主字体**: Roboto / Helvetica Neue
- **标题**: 粗体,字重700
- **正文**: 常规,字重400
- **代码**: Fira Code / Consolas

---

### 色彩规范

#### 主色调
| 颜色名称 | 色值 | 用途 |
|---------|------|------|
| 冷灰 | #333333 | 主背景 |
| 科技蓝 | #0099FF | 主强调色 |
| 深蓝 | #0066CC | 辅助强调色 |
| 纯黑 | #000000 | 三体界面背景 |

#### 功能色
| 颜色名称 | 色值 | 用途 |
|---------|------|------|
| 红色 | #FF0000 | 警告/威慑触发 |
| 绿色 | #00FF00 | 正常/稳定 |
| 黄色 | #FFFF00 | 警告/波动 |
| 青绿 | #00FFCC | 代码/数据流 |
| 白色 | #FFFFFF | 文字/内容 |

---

### 动画效果

#### 转场动画
- **淡入淡出**: 场景切换,0.5秒
- **滑动**: 侧边栏展开,0.3秒
- **缩放**: 弹窗出现,0.2秒

#### 元素动画
- **脉冲**: 状态指示灯,1秒周期
- **闪烁**: 警告信息,0.5秒周期
- **滚动**: 数据流,匀速持续
- **波形**: 频率分析,实时更新

---

### 特效规范

#### 光晕效果
- **应用**: 按钮、指示灯、关键元素
- **参数**:
  - 颜色: 元素本色
  - 扩散半径: 10-20px
  - 透明度: 0.3-0.5

#### 扫描线
- **应用**: 三体界面,增强科技感
- **参数**:
  - 颜色: 黑色半透明
  - 间距: 2px
  - 速度: 静止

#### 故障效果
- **应用**: 系统异常,增强紧张感
- **参数**:
  - 颜色偏移: RGB通道分离
  - 位置偏移: 随机2-5px
  - 频率: 10Hz

## 技术实现建议

### 图层管理
- **背景层**: 纯色或渐变背景
- **数据层**: 滚动文字、波形图
- **UI层**: 按钮、指示灯、边框
- **特效层**: 光晕、扫描线、故障效果
- **遮罩层**: 视窗、圆形遮罩

### 性能优化
- **数据流**: 使用缓存预渲染
- **波形图**: 降低采样率,减少计算
- **动画**: 使用CSS硬件加速
- **字体**: 使用Web字体预加载

### 兼容性
- **分辨率**: 1920x1080 (16:9)
- **帧率**: 30fps
- **编码**: H.264
- **格式**: PNG/JPEG (静态), MP4 (动态)

---

## 使用指南

1. **基准设计**: 先生成核心元素的基准图
2. **一致性**: 保持同一元素在不同场景的风格一致
3. **适应性**: 根据分镜需求调整UI元素大小和位置
4. **动画**: 动画效果必须符合场景情绪和节奏
5. **测试**: 在实际视频中测试UI的可读性和视觉效果

**备注**: 所有UI设计必须符合《三体》原著设定,避免过度装饰,保持极简主义和功能主义美学。
