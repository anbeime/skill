## 📝 配置步骤(3步完成)

### 步骤1: 获取Tushare令牌(免费)

1. **注册账户**

2. **获取令牌**

### 步骤2: 配置小易(Claude Desktop)

#### 找到配置文件

**Windows系统**:
```
%APPDATA%\Claude\claude_desktop_config.json
```
快捷方式: 按 `Win+R`，输入上面的路径，回车

**macOS系统**:
```
~/Library/Application Support/Claude/claude_desktop_config.json
```

#### 编辑配置文件

用记事本或VS Code打开配置文件，添加以下内容:

```json
{
  "mcpServers": {
    "finance-mcp": {
      "type": "stdio",
      "command": "finance-mcp",
      "args": [],
      "env": {
        "TUSHARE_TOKEN": "这里替换成你的Tushare令牌"
      }
    }
  }
}
```

**注意**: 
- 将 `这里替换成你的Tushare令牌` 替换为步骤1获取的Token
- 确保JSON格式正确(注意逗号和引号)

## ⚠️ 常见问题

### Q1: 配置后Claude没有反应?
**A**: 
1. 检查配置文件路径是否正确
2. 检查JSON格式(可用在线JSON验证工具)
3. 确认Tushare令牌是否正确
4. 完全关闭并重启Claude
5. 查看Claude日志: `%APPDATA%\Claude\logs`

### Q2: 提示"Tushare token source: none"?
**A**: 
- 令牌未正确配置
- 检查配置文件中的`TUSHARE_TOKEN`字段
- 确保没有多余的空格或引号

### Q3: 查询某些数据提示"积分不足"?
**A**: 
- Tushare部分高级数据需要积分
- 基础数据(股价、K线)免费
- 学生可申请2000免费积分
- 或升级Tushare会员

### Q4: 数据有延迟?
**A**: 
- 免费账户数据可能有15分钟延迟
- 升级Tushare会员可获取实时数据

### Q5: 想使用公共云服务(不配置令牌)?
**A**: 
修改配置为:
```json
{
  "mcpServers": {
    "finance-mcp-cloud": {
      "type": "streamableHttp",
      "url": "https://finvestai.top/mcp",
      "timeout": 600
    }
  }
}
```
注意: 公共服务可能有速率限制
