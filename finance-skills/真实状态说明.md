## 如何真正让小易拥有金融能力？

### 方案1: 确认小易是否支持MCP

**第一步：检查小易的技能配置方式**

小易(StepFun AI助手)可能有以下几种配置方式：

#### A. 如果支持MCP协议
需要找到配置文件，可能在:
- `%APPDATA%\StepFun\config.json`
- `%USERPROFILE%\.stepfun\config.json`
- `D:\tool\skill\` 目录下的某个配置文件

#### B. 如果使用Skill系统
可能需要将技能放到特定目录:
- `D:\tool\skill\` (你的skill目录)
- 每个skill需要有 `SKILL.md` 文件

#### C. 如果使用插件系统
可能需要特定的插件格式

### 方案2: 使用已有的公共服务(最简单)

**不需要配置，直接使用Web版:**

访问: https://finvestai.top/

这是finance-mcp提供的免费在线服务，可以直接在网页中:
- 查询股票数据
- 计算技术指标
- 获取财务报表
- 搜索财经新闻

### 方案3: 手动调用finance-mcp命令

虽然不能集成到小易，但可以在命令行中使用:

```bash
# 设置环境变量
set TUSHARE_TOKEN=你的令牌

# 启动服务
finance-mcp
```

然后通过其他支持MCP的工具(如Claude Desktop、Cursor等)使用。
