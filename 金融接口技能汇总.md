## 🌟 重点推荐项目

### 1️⃣ FinanceMCP ⭐⭐⭐⭐⭐
**项目地址**: https://github.com/guangxiangdebizi/FinanceMCP  
**Stars**: 437+ | **Forks**: 95+

#### 核心特性
- ✅ **基于MCP协议的专业金融数据服务器**
- ✅ **集成Tushare API + Binance API**
- ✅ **支持14个工具** + 5大技术指标(MACD/RSI/KDJ/BOLL/MA)
- ✅ **覆盖10大市场**: A股、美股、港股、外汇、期货、基金、债券、期权、加密货币
- ✅ **智能技术指标引擎**: 自动预取历史数据，消除NaN值
- ✅ **提供免费公共云服务**: https://finvestai.top/

#### 功能清单
| 工具名称 | 功能描述 | 核心特色 |
|---------|---------|---------|
| `current_timestamp` | 当前时间戳 | UTC+8时区 |
| `finance_news` | 财经新闻搜索 | 百度新闻爬虫 |
| `stock_data` | 股票/加密+技术指标 | 10大市场+5技术指标 |
| `index_data` | 指数数据 | 主要市场指数历史 |
| `csi_index_constituents` | CSI指数成分与权重 | 估值/财务指标 |
| `macro_econ` | 宏观经济数据 | 11指标(GDP/CPI/PPI/PMI等) |
| `company_performance` | A股公司财务分析 | 13数据类型 |
| `company_performance_hk` | 港股公司财务分析 | 3大财务报表 |
| `company_performance_us` | 美股公司财务分析 | 4大财务报表 |
| `fund_data` | 基金数据 | 净值/持仓/分红 |
| `fund_manager_by_name` | 基金经理查询 | 个人背景、管理基金 |
| `convertible_bond` | 可转债数据 | 基本信息+转换条款 |
| `block_trade` | 大宗交易数据 | 交易详情+对手信息 |
| `money_flow` | 资金流向数据 | 主力/超大单/大单分析 |
| `margin_trade` | 融资融券数据 | 标的股票/汇总/明细 |
| `dragon_tiger_inst` | 龙虎榜机构明细 | 买卖额/比例/净额 |
| `hot_news_7x24` | 7×24热点 | 相似度80%去重 |

#### 技术指标
- **MACD** `macd(12,26,9)` - 趋势分析
- **RSI** `rsi(14)` - 超买超卖判断
- **KDJ** `kdj(9,3,3)` - 随机指标
- **BOLL** `boll(20,2)` - 布林带
- **MA** `ma(5/10/20/60)` - 移动平均线

#### 安装方式
```bash
# 方式1: NPM全局安装(推荐)
npm install -g finance-mcp

# 方式2: 使用公共云服务(免费)
# 在Claude配置文件中添加:
{
  "mcpServers": {
    "finance-mcp": {
      "type": "streamableHttp",
      "url": "https://finvestai.top/mcp",
      "headers": {
        "X-Tushare-Token": "您的tushare令牌"
      }
    }
  }
}
```

### 3️⃣ Fin-Agent ⭐⭐⭐⭐
**项目地址**: https://github.com/YUHAI0/fin-agent  
**技术栈**: DeepSeek V3 + Tushare Pro

#### 核心特性
- ✅ **全自动金融领域数据分析及股票推荐智能体**
- ✅ **基于DeepSeek V3大模型** - 强大逻辑推理能力
- ✅ **集成Tushare Pro金融数据** - 实时股价、K线、财务指标
- ✅ **智能任务拆解** - 自动规划复杂分析流程
- ✅ **一键安装** - `pip install fin-agent`

#### 功能亮点
1. **拒绝幻觉，数据为王** - 直接调用Tushare接口获取真实数据
2. **智能规划** - 自动拆解复杂任务为多步骤执行
3. **极简交互** - 自然语言提问，秒级返回分析结果

#### 安装使用
```bash
# 1. 一键安装
pip install fin-agent

# 2. 启动对话
fin-agent

# 3. 首次运行配置DeepSeek API Key和Tushare Token
```

#### 示例查询
```
"帮我分析一下万科A现在的估值水平，对比一下它去年的利润情况。"
"推荐几只最近涨势好的银行股，并分析它们的基本面。"
```

```bash
# 1. 安装FinanceMCP
npm install -g finance-mcp

# 2. 获取Tushare Token
# 访问 https://tushare.pro/register 注册并获取

# 3. 配置Claude Desktop
# 编辑配置文件(Windows: %APPDATA%\Claude\claude_desktop_config.json)

  "mcpServers": {
    "finance-mcp": {
      "type": "stdio",
      "command": "finance-mcp",
      "env": {
        "TUSHARE_TOKEN": "你的token"

    }
  }
}

# 4. 重启Claude Desktop
# 5. 开始提问!
```

## 📊 数据覆盖范围

### 市场类型
- ✅ **A股市场** - 沪深主板、创业板、科创板、北交所
- ✅ **港股市场** - 恒生指数成分股、主板、创业板
- ✅ **美股市场** - 纳斯达克、纽交所
- ✅ **加密货币** - Bitcoin、Ethereum、USDT等主流币种
- ✅ **基金市场** - 公募基金、ETF、货币基金
- ✅ **债券市场** - 可转债、国债、企业债
- ✅ **期货市场** - 商品期货、股指期货
- ✅ **外汇市场** - 主要货币对

### 数据类型
- ✅ **实时行情** - 股价、成交量、涨跌幅
- ✅ **历史数据** - K线、分时、复权数据
- ✅ **财务数据** - 利润表、资产负债表、现金流量表
- ✅ **技术指标** - MACD、RSI、KDJ、BOLL、MA
- ✅ **资金流向** - 主力资金、大单、中单、小单
- ✅ **宏观数据** - GDP、CPI、PPI、PMI、Shibor
- ✅ **新闻资讯** - 财经新闻、公告、研报
- ✅ **情绪分析** - 市场情绪、舆情监控

## ⚠️ 注意事项

### 数据使用
1. **Tushare积分限制** - 部分高级数据需要积分
2. **API调用频率** - 注意接口调用限制
3. **数据延迟** - 免费数据可能有15分钟延迟
4. **数据准确性** - 仅供参考，不构成投资建议
