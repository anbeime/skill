# 小跃技能对话小助手

这是一个整合了多个小跃助手项目的技能仓库，旨在为用户提供全面的对话助手能力。

## 📦 项目结构

```
skill/
├── companion-skill/     # 小跃伴侣技能版（TypeScript）
├── companion-simple/    # 小跃伴侣简化版（Python）
├── assistant/           # 小跃助手核心（TypeScript）
└── README.md           # 本文件
```

## 🚀 子项目介绍

### companion-skill
基于 TypeScript 的小跃伴侣技能版本，提供：
- 场景检测能力
- 图像生成集成
- 个性化对话提示
- 完整的技能系统

**快速开始**: 查看 [companion-skill/README.md](./companion-skill/README.md)

### companion-simple
基于 Python 的小跃伴侣简化版本，提供：
- 技能数据爬取与管理
- Web 界面展示
- API 服务集成
- 定时任务调度

**快速开始**: 查看 [companion-simple/README.md](./companion-simple/README.md)

### assistant
小跃助手核心系统，提供：
- 智能代理框架
- 记忆管理系统
- 文件管理技能
- 多平台支持

**快速开始**: 查看 [assistant/README.md](./assistant/README.md)

## 📖 使用指南

每个子项目都有独立的文档和配置：

1. **companion-skill**: 适合需要高级技能系统的场景
2. **companion-simple**: 适合快速部署和 Web 展示
3. **assistant**: 适合构建完整的助手应用

## 🔧 开发环境

- **TypeScript 项目**: Node.js 16+ 
- **Python 项目**: Python 3.8+
