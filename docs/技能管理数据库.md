## 🔧 API配置指南

### Google Gemini API（最常用）

**获取方式**：
1. 访问 https://ai.google.dev/
2. 点击 "Get API Key"
3. 创建新项目或选择现有项目
4. 生成API密钥

**配置方式**：
```bash
# 在技能目录创建 .env 文件
GEMINI_API_KEY=your_api_key_here
```

**免费额度**：
- 每分钟60次请求
- 每天1500次请求
- 完全免费

**适用技能**（17个）：
- content-creation-publisher
- intelligent-content-system
- article-illustrator
- baoyu-xhs-images
- wechat-hotspot-publisher
- NanoBanana-PPT-Skills
- ppt-generator
- xiaohongshu-makeup
- pop-up-book-illustration
- poetry-music-visual
- 以及其他7个技能

### 可灵AI API（可选）

**获取方式**：
1. 访问 https://www.kling.ai/
2. 注册账号并实名认证
3. 进入开发者中心
4. 创建应用并获取密钥

**配置方式**：
```bash
# 在技能目录创建 .env 文件
KLING_ACCESS_KEY=your_access_key_here
KLING_SECRET_KEY=your_secret_key_here
```

**费用**：
- 按使用量计费
- 视频生成约0.5-2元/次
